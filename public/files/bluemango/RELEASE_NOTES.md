# BlueMango Release Notes

## v0.9.0 (2026-02-20)

### 🧪 테스트
- **E2E 테스트 인프라 구축** — DB 스키마 격리 기반 E2E 테스트 환경 (#59)
- **E2E 전체 통과** — 342건 테스트 전체 통과 (DB 스키마 격리 적용)
- **어드민 스모크 테스트** — 27개 전체 페이지 스모크 E2E 테스트 추가
- **E2E 테스트 실패 수정** — 어드민/프론트엔드 E2E 테스트 안정화

### 🐛 버그 수정
- **여행자 정보 모달** — 모바일 스크롤 안 되는 버그 수정 (재수정 포함)
- **알림톡 API** — URL 수정 및 버튼 URL을 bizmsg 등록값으로 고정
- **검색** — keyword 파라미터 지원 추가 (q 또는 keyword 둘 다 허용)
- **Admin API** — Users/Settlements API 파라미터 불일치 수정

### ♻️ 리팩토링
- **알림톡 템플릿** — 서비스 리팩토링 및 미사용 템플릿 삭제
- **이메일 템플릿** — 서비스 리팩토링 및 미사용 템플릿 삭제

### 📄 문서

- 기능 명세서 CSV 변환 (어드민 35건, 프론트엔드 41건)
