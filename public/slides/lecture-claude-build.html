📄 전체 강의 자료 페이지 보기 → — 세로 스크롤 풀 버전(준비물·여정·커리큘럼·도구 한눈에)

개요

항목내용
제목클로드로 만드는 나만의 지식노트 + 배포되는 내 사이트
형식핸즈온 + 개인교습 (4시간 / 240분)
대상비개발 실무자 (프롬프트·코딩 경험 무관)
강사코드몬 (codemon.ai)
최종 산출물① 클로드가 읽는 내 지식노트 볼트   ② 인터넷에 배포된 내 사이트 URL

이런 분에게 추천합니다

  • AI를 써보고 싶은데 어디서부터 시작해야 할지 모르겠는 분
  • 메모는 쌓는데 다시 안 보게 되는 분 — AI가 읽고 정리해주는 지식노트가 필요한 분
  • 코드 한 줄 몰라도 내 사이트를 인터넷에 띄워보고 싶은 분
  • 클로드를 ‘대화’를 넘어 ‘일 맡기는 도구’ 로 쓰고 싶은 분

핵심 성과

4시간 후, 손에 동작하는 결과물 2개를 들고 나갑니다.

  • 클로드가 읽고 요약·검색·정리해주는 내 지식노트 볼트
  • 친구에게 바로 보낼 수 있는, 인터넷에 떠 있는 내 사이트 URL
  • 혼자서도 이어갈 30일 자립 로드맵

준비물

준비물설명비용
클로드 유료 플랜Claude Code·Cowork는 유료 전용. 안정성 위해 Max 권장(5x $100 / 20x $200·월). 네이티브 설치라 Node 없이도 설치됨유료
클로드 데스크탑 앱맥/윈도우 공통. Cowork(터미널 없이 파일 다루는 에이전트 모드) 실행—
옵시디언(Obsidian)지식노트 ‘볼트’(.md 폴더). 변환 없이 클로드가 그대로 읽음무료
Node.js일부 도구 연결·사이트 빌드에 사용 (nodejs.org/ko/download)무료
안티그래비티 IDE구글 에이전트-우선 IDE (antigravity.google). 무료 사용량 제한 있어 보조로 사용무료
GitHub · Vercel · SupabaseGitHub는 구글로 가입 → Vercel은 그 GitHub로 가입 → Supabase 가입무료

다 못 깔아도 괜찮습니다 — 0교시(20분) 에 강사와 함께 세팅합니다.

커리큘럼 (240분)

시연 20% · 직접 실습 60% · 산출물 확인 20% — 각 교시 끝에 손에 잡히는 결과 하나씩.

0교시. 세팅 & 워밍업 (20분)

  • 오늘 만들 산출물 2개 완성본 미리보기
  • 준비물 점검·설치 트러블슈팅, 계정 연결(GitHub→Vercel→Supabase)
  • 안전 수칙 — ‘넣지 말 것’ 레드리스트

산출물 — 전원 세팅 완료

1교시. LLM 위키 = 나만의 지식 노트 (60분)

  • (개념 10’) 마크다운 볼트 + 백링크 [[ ]] = 변환 없이 그대로 읽히는 지식창고
  • (실습 40’) 옵시디언 볼트 생성 → 노트 3~5개 + 백링크 → 클로드 파일시스템 커넥터로 폴더 연결(Settings → Connectors → Filesystem)
  • 클로드에게 내 볼트 요약·검색·새 노트 생성·정리 위임

산출물 — 내 지식 볼트 + 정리 노트 1개

2교시. Skill·MCP 개념 + Claude·Cowork 핸즈온 (60분)

  • (개념 15’) Skill = 클로드에게 ‘하는 법’을 패키지로 가르침(PDF·엑셀·PPT 내장 스킬) / MCP = 외부 도구·로컬 파일에 잇는 통로 / Cowork = 터미널 없이 에이전트로 파일 작업
  • (실습 35’) 내장 Skill 체감(PDF·엑셀 자동 처리) → 커넥터/MCP 원클릭 추가 → Cowork로 폴더 통째 작업해 실파일 산출

산출물 — 내 커넥터·스킬로 만든 산출물

3교시. 실제 배포되는 사이트 만들기 ★핵심 (90분)

  • (설계 15’) 만들 사이트(프로필 + 즉흥 주제) · 스택 그림 GitHub→Vercel→Supabase · 무료 한도 고지
  • (빌드 60’) Vercel ‘Supabase Starter’ Deploy → 환경변수 자동 주입 → 라이브 URL 발급 → 클로드 코드/안티그래비티 에이전트로 내용·디자인 수정 → git push → 자동 재배포 → 즉흥 주제 페이지 변형

산출물 — 인터넷에 떠 있는 내 사이트 URL

4교시. 마무리 & 자립 (10분)

  • 산출물 2개 회고 + 라이브 URL 공유
  • 30일 자립 로드맵 · 비용 정리 · Q&A

산출물 — 30일 자립 플랜

사용 도구 — 가장 단순·안정한 경로

4시간 안에 ‘실제 동작’까지 가려면 길 선택이 전부입니다. 모두 공식 문서로 확인한 경로만 채택했습니다.
