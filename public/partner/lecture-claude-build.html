4시간 핸즈온 비개발자 대상 개인교습형 실제 프로덕트 배포

클로드로 만드는나만의 지식노트 + 배포되는 내 사이트

코드 한 줄 몰라도 괜찮습니다. 클로드에게 일을 맡겨 흩어진 메모를 AI가 읽는 지식노트로 만들고, 끝나면 인터넷에 실제로 떠 있는 내 사이트 주소 하나를 손에 쥐고 나가는 4시간입니다.

오늘 손에 남는 것

말이 아니라, 동작하는 결과물 2개

01

내 지식노트 볼트

Obsidian 마크다운 볼트 + 클로드 연결. 내가 쌓은 메모를 클로드가 읽고 요약·검색·정리해 주는, 살아있는 제2의 뇌.

02

배포된 내 사이트

Vercel + Supabase로 인터넷에 실제로 올라간 프로필/주제 사이트. 친구에게 바로 보낼 수 있는 라이브 URL.

시작 전 준비물

이참에 세팅부터 끝내시죠

아래는 미리 깔아 오면 좋은 것들. 못 하셔도 괜찮습니다 — 수업 0교시에 같이 합니다.

🟣 클로드 (Claude) 유료 플랜

Claude Code·Cowork는 유료 플랜에서만 동작합니다. 안정적으로 가려면 Max 권장(Max 5x $100/월 · 20x $200/월). 네이티브 설치라 Node.js 없이도 깔립니다.

유료 · Max 권장

💬 클로드 데스크탑 앱

맥/윈도우 공통. Cowork(터미널 없이 파일을 직접 다루는 에이전트 모드)가 여기서 돕니다. 앱은 켜둔 채로 작업.

맥 / 윈도우

🪨 옵시디언 (Obsidian)

설치 필수. 우리 지식노트의 '볼트'가 됩니다. 그냥 마크다운(.md) 폴더라 별도 변환 없이 클로드가 그대로 읽습니다.

무료

🟩 Node.js

일부 도구 연결(파일시스템 MCP)과 사이트 빌드에 쓰입니다. nodejs.org/ko/download 에서 설치.

무료

🚀 안티그래비티 IDE

구글의 에이전트-우선 IDE(무료). 에이전트가 계획→실행→검증을 보여줍니다. antigravity.google · ⚠️무료 사용량 제한이 있어 수업에선 보조로 사용.

무료 · 보조

🔗 GitHub · Vercel · Supabase 가입

GitHub는 구글로 가입 → Vercel은 그 GitHub로 가입 → Supabase 가입. 사이트 배포 3종 세트입니다.

무료

위 준비물을 다 못 깔아도 괜찮습니다. 수업 0교시(20분)에서 강사와 1:1로 함께 세팅합니다.

1

지식을 쌓는다

옵시디언에 메모를 모으고 클로드를 연결해 'AI가 읽는 지식노트'로.

2

일을 맡긴다

Skill·MCP·Cowork로 클로드에게 파일 작업·도구 연결을 위임.

Vercel+Supabase로 내 사이트를 인터넷에 실제 배포.

1교시 · 60분

LLM 위키 = 나만의 지식 노트

  • (개념 10') 마크다운 볼트 + 백링크 [[ ]] = 변환 없이 그대로 읽히는 지식창고
  • (실습 40') 옵시디언 볼트 생성 → 노트 3~5개 + 백링크 → 클로드 파일시스템 커넥터로 폴더 연결(Settings→Connectors→Filesystem)
  • 클로드에게 내 볼트 요약·검색·새 노트 생성·정리 위임
내 지식 볼트 + 정리 노트 1개
2교시 · 60분

Skill · MCP 개념 + Claude · Cowork 핸즈온

  • (개념 15') Skill = 클로드에게 '하는 법'을 패키지로 가르침(PDF·엑셀·PPT 내장 스킬) / MCP = 외부 도구·로컬 파일에 잇는 통로 / Cowork = 터미널 없이 에이전트로 파일 작업
  • (실습 35') 내장 Skill 체감(PDF·엑셀 자동 처리) → 커넥터/MCP 원클릭 추가 → Cowork로 폴더 통째 작업해 실파일 산출
내 커넥터·스킬로 만든 산출물

실제 배포되는 사이트 만들기

  • (설계 15') 만들 사이트(프로필 + 즉흥 주제) · 스택 그림 GitHub→Vercel→Supabase · 무료 한도 고지
  • (빌드 60') Vercel 'Supabase Starter' Deploy → GitHub clone + 환경변수 자동 주입 → 라이브 URL 발급
  • 클로드 코드/안티그래비티 에이전트로 내용·디자인 수정 → git push → 자동 재배포 → 즉흥 주제 페이지 변형
