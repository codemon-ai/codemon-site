# Codemon 서비스 문서

## 개요

Codemon은 다양한 모니터링, 신호, 유틸리티 서비스를 운영합니다.
각 서비스는 독립적으로 작동하지만 공통 인프라(PostgreSQL, PM2, Ngrok)를 공유합니다.

## 서비스 분류

### 🔧 모니터링 & 관리
- **[Mini Admin](/docs/mini-admin)**: PM2 프로세스 통합 관리 대시보드
- **[Slimmon](/docs/slimmon)**: 경량 시스템 모니터링 서비스
- **[Jipmon](/docs/jipmon)**: 지정 모니터링 서비스
- **[Memomon](/docs/memomon)**: 메모리 관리 서비스

### 📊 신호 & 알림
- **[Dollar Signal](/docs/dollar-signal)**: 주식/암호화폐 가격 신호 분석

### 🎨 엔터테인먼트 & 도구
- **[Tarotmon](/docs/tarotmon)**: AI 타로 카드 리딩 서비스
- **[ASCII Art](/docs/ascii-art)**: 아스키 아트 생성 도구
- **[Forest99](/docs/forest99)**: 3D 웹 게임 (Three.js)

### 📝 이 사이트
- **[Codemon Site](/docs/codemon-site)**: Tech Lead 기술 블로그 & 문서 사이트

## 기술 스택 개요

| 카테고리 | 기술 |
|---------|------|
| **Backend** | FastAPI (Python), Express (Node.js) |
| **Frontend** | Next.js 14/16, React 18/19 |
| **Database** | PostgreSQL |
| **Styling** | Tailwind CSS |
| **3D Graphics** | Three.js |
| **Process Manager** | PM2 |
| **Tunneling** | Ngrok |

## 빠른 시작

각 서비스의 개별 문서 페이지에서 로컬 실행, 아키텍처, API 등 상세 정보를 확인하세요.

### 전체 서비스 시작

```bash
# PM2로 모든 자동시작 서비스 실행 (19개)
pm2 start ecosystem.config.js

# 상태 확인
pm2 list
```

### 개별 서비스 시작

```bash
# 특정 서비스만 시작
pm2 start mini-admin
pm2 start dollar-signal-api
pm2 start tarotmon
```

## 공통 인프라

### PM2 Ecosystem
- **경로**: `/Users/coffeemon/workspace/codemon/ecosystem.config.js`
- **관리**: [Mini Admin](https://mini-admin.ngrok.io) 웹 UI
- **자동시작**: 19개 서비스

### PostgreSQL
대부분의 서비스가 공유하는 데이터베이스:
- **Host**: localhost:5432
- **사용 서비스**: dollar-signal, jipmon, tarotmon 등

### Ngrok 터널
각 주요 서비스는 Ngrok 터널을 통해 외부 접근 가능:
- `ngrok-mini-admin` → https://mini-admin.ngrok.io
- `ngrok-dollar-signal-api` → 외부 API 접근
- `ngrok-tarotmon` → 외부 웹 접근
- 등등...
