## 프로젝트 문서

각 프로젝트의 상세 기술 문서입니다.

- **그림하루**: AI 그림일기 서비스 개발 문서
- **타로몬**: AI 타로 리딩 서비스 개발 문서
- **달러시그널**: 환율 예측 모델 문서

> 문서는 지속적으로 업데이트됩니다.

개발 과정에서 정리한 기술 가이드입니다.

- OpenAI API 활용
- Vercel 배포
