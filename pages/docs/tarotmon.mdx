# Tarotmon

## 목적

Tarotmon은 AI를 활용한 타로 카드 리딩 서비스입니다.
사용자의 질문에 대해 타로 카드를 뽑고 AI가 해석을 제공합니다.

## 기술 스택

| 카테고리 | 기술 |
|---------|------|
| **Framework** | Next.js |
| **Runtime** | Node.js |
| **UI** | React, Tailwind CSS |
| **Port** | 3010 |

## 주요 기능

### 1. 타로 카드 리딩
- 랜덤 카드 뽑기
- 3장/5장/10장 스프레드
- 카드 의미 해석

### 2. AI 해석
- 질문 기반 리딩
- 카드 조합 분석
- 개인화된 메시지

### 3. UI/UX
- 애니메이션 효과
- 다크모드 지원
- 모바일 최적화

## 시작하기

### 로컬 실행

```bash
cd ~/workspace/codemon/tarotmon

# 개발 모드
npm run dev

# 프로덕션 모드 (PM2)
pm2 start tarotmon
```

### 접속 방법

**로컬**:
```
http://localhost:3010
```

**외부 접근** (Ngrok):
```
https://tarotmon.ngrok.io
```
