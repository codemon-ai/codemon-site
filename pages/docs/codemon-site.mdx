# Codemon Site

## 목적

Codemon Site는 Tech Lead 관점의 기술 블로그와 프로젝트 문서를 제공하는 사이트입니다.
Nextra 기반으로 구축되어 빠르고 SEO 친화적인 콘텐츠를 제공합니다.

## 기술 스택

| 카테고리 | 기술 |
|---------|------|
| **Framework** | Nextra 3.3.x (nextra-theme-docs) |
| **Runtime** | Next.js 14 (Pages Router) |
| **Language** | TypeScript |
| **Styling** | Tailwind CSS |
| **Port** | 3002 |

## 프로젝트 구조

```
codemon-site/
├── pages/              # Pages Router
│   ├── _app.tsx        # App wrapper
│   ├── _meta.ts        # Top-level navigation
│   ├── index.mdx       # Brand landing page
│   ├── about.mdx       # About page
│   ├── projects/       # Projects section
│   ├── blog/           # Blog section
│   ├── docs/           # Documentation section
│   └── p/              # Private pages (비공개)
├── components/         # React components
│   ├── Hero.tsx        # Landing hero section
│   └── Features.tsx    # Features cards
├── theme.config.tsx    # Nextra theme config
├── next.config.mjs     # Next.js config
└── tailwind.config.js  # Tailwind config
```

## 주요 섹션

### 1. 블로그 (`/blog`)
- Tech Lead 관점의 기술 글
- 프로젝트 회고록
- 아키텍처 결정 기록

### 2. 프로젝트 (`/projects`)
- 15개 프로젝트 문서
- 기술 스택 상세 설명
- 라이브 데모 링크

### 3. 문서 (`/docs`)
- 서비스 문서 (이 페이지)
- 아키텍처 가이드
- 개발 환경 설정

### 4. 비공개 페이지 (`/p`)
- UUID 기반 비공개 URL
- 네비게이션에 노출 안 됨
- robots.txt로 크롤링 방지

## 비공개 페이지 시스템

### 구조
```
pages/p/
├── _meta.ts              # 심플 레이아웃 설정
└── {uuid}.mdx            # 비공개 콘텐츠
```

### 발행 워크플로우
1. Obsidian에서 마크다운 작성
2. Claude에게 "비공개 페이지로 발행해줘" 요청
3. 자동 UUID 생성 및 파일 생성
4. git commit & push
5. Vercel 자동 배포
6. URL 공유: `https://codemon.ai/p/{uuid}`

### 핵심 파일

**pages/p/_meta.ts**:
```typescript
export default {
  '*': {
    theme: {
      sidebar: false,
      toc: false,
      breadcrumb: false,
      pagination: false,
    },
  },
}
```

**public/robots.txt**:
```
User-agent: *
Disallow: /p/

Sitemap: https://codemon.ai/sitemap.xml
```

## 시작하기

### 로컬 실행

```bash
cd /Users/coffeemon/workspace/codemon/brand/codemon-site

# 개발 모드
npm run dev

# 프로덕션 빌드
npm run build

# 프로덕션 서버 (PM2)
pm2 start codemon-site
```

**로컬**:

## 컨벤션

### 페이지 레이아웃
- `index` 페이지: `layout: 'raw'` (no sidebar/TOC)
- 다른 페이지: 표준 Nextra docs 레이아웃

### 컴포넌트
- React 함수형 컴포넌트 + TypeScript
- Tailwind CSS로 스타일링
- 다크모드 `dark:` 변형 지원

### 색상
- Primary: Blue gradient (#3b82f6 → #8b5cf6)
- Background: Light (#ffffff) / Dark (#0a0a0a)
