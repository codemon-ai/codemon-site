# ASCII Art

## 목적

ASCII Art는 텍스트 기반 아스키 아트를 생성하는 도구입니다.
이미지를 아스키 문자로 변환하거나 텍스트 배너를 만들 수 있습니다.

## 기술 스택

| 카테고리 | 기술 |
|---------|------|
| **Backend** | Node.js, Express |
| **Server** | web/server.js |
| **Port** | 3030 |

## 주요 기능

### 1. 이미지 → 아스키 변환
- PNG, JPG 이미지를 아스키 아트로 변환
- 해상도/밀도 조절
- 컬러/흑백 선택

### 2. 텍스트 배너 생성
- 다양한 폰트 스타일
- 크기 조절
- 테두리 옵션

### 3. 내보내기
- 텍스트 파일 다운로드
- 클립보드 복사
- 이미지로 저장

## 아스키 문자 세트

### 기본 세트
```
 .:-=+*#%@
```

### 확장 세트
```
 .'`^",:;Il!i><~+_-?][}{1)(|/tfjrxnuvczXYUJCLQ0OZmwqpdbkhao*#MW&8%B@$
```

## API 엔드포인트

```
POST /api/convert/image   # 이미지 변환
POST /api/convert/text    # 텍스트 배너
GET  /api/fonts           # 사용 가능한 폰트 목록
```
