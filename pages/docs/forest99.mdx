# Forest99

## 목적

Forest99는 Three.js로 만든 3D 웹 게임입니다.
브라우저에서 바로 플레이할 수 있는 인터랙티브 3D 경험을 제공합니다.

## 기술 스택

| 카테고리 | 기술 |
|---------|------|
| **3D Engine** | Three.js |
| **Framework** | React |
| **Build Tool** | Webpack/Vite |
| **Styling** | CSS3, WebGL |

## 주요 기능

### 1. 3D 렌더링
- Three.js 기반 3D 그래픽
- 실시간 조명 효과
- 텍스처 매핑

### 2. 인터랙션
- 마우스/키보드 컨트롤
- 물리 엔진 적용
- 충돌 감지

### 3. 게임 메커니즘
- 캐릭터 이동
- 오브젝트 상호작용
- 점수 시스템

## 시작하기

```bash

# 개발 모드

```

### 접속 방법

```

```

## 3D 에셋

### 모델
- GLTF/GLB 포맷
- FBX 변환 지원

### 텍스처
- Diffuse maps
- Normal maps
- PBR materials

### 오디오
- 배경 음악
- 효과음

## 성능 최적화

### LOD (Level of Detail)
- 거리에 따른 모델 단순화
- 오브젝트 컬링
- 텍스처 압축

### 렌더링 최적화
- Frustum culling
- Occlusion culling
- 인스턴싱

## 컨트롤

### 키보드
- `W/A/S/D`: 이동
- `Space`: 점프
- `E`: 상호작용

### 마우스
- 좌클릭: 선택/공격
- 우클릭: 카메라 회전
- 휠: 줌 인/아웃
