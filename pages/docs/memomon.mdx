# Memomon

## 목적

Memomon은 시스템 메모리를 효율적으로 관리하는 서비스입니다.
메모리 사용 패턴을 분석하고 최적화 제안을 제공합니다.

## 기술 스택

| 카테고리 | 기술 |
|---------|------|
| **Framework** | Next.js |
| **Runtime** | Node.js |
| **UI** | React, Tailwind CSS |

## 주요 기능

### 1. 메모리 모니터링
- 실시간 메모리 사용량 추적
- 프로세스별 메모리 분석
- 메모리 누수 감지

### 2. 최적화 제안
- 메모리 해제 권장사항
- 프로세스 우선순위 조정
- 캐시 관리 팁

### 3. 히스토리 분석
- 시간대별 메모리 사용 패턴
- 피크 타임 분석
- 트렌드 차트

## 시작하기

### 로컬 실행

```bash
cd /Users/coffeemon/workspace/codemon/memomon

# 개발 모드
npm run dev

# 프로덕션 모드 (PM2)
pm2 start memomon
```

### 접속 방법

**로컬**:
```
http://localhost:{port}
```

- **실시간 스캔**: 5초마다 메모리 상태 체크
- **임계값 알림**: 사용률 80% 이상 시 경고
- **자동 정리**: 설정된 조건에 따라 캐시 정리
