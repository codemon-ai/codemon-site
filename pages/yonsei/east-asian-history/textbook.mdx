# 주교재 정리 — 19~66쪽 (제1장~제4장)

## 핵심 키워드

중국의 다원성, 화이사상(華夷思想), 주권국가, 내셔널리즘, 량치차오, 황허문명, 갑골문, 봉건제(封建制), 군현제(郡縣制), 춘추전국시대, 진시황 통일, 흉노, 묵특선우, 백등산 전투, 화친, 장건, 중행열(中行說), 무제, 하서사군, 소금철 전매, 균수법/평준법, 천명관, 왕망, 5호16국, 호한체제, 북위 효문제, 한화정책, 육조정권, 강남개발, 토단법, 조공/책봉, 동아시아 세계론

## 제1장: 중국이란? (pp. 19-28)

### 1. '중국' 속의 다원성
- 중화인민공화국: 약 **960만 km²**, 세계 3~4위 면적, 인구 약 **13억**
- **56개 민족**: 한족(92%) + **55개 소수민족**
- 인구 분포: 동부 해안에 집중, 서부(티베트 고원 등)는 1㎢당 10명도 안 되는 지역
- 황허(黃河)와 양쯔강(長江) 두 강의 대하(大河)를 중심으로 남북 구분

### 2. '중국' 개념의 형성
- **기원전 10세기부터 사용**: 가요를 모은 「시경(詩經)」에 이미 등장
- 초기 '중국': 왕이 거주하는 **수도 주변의 좁은 지역** (도시국가)
- **전국시대**: '중국에 군림하고' '중국을 금하고' 등 표현 → 통일성을 가진 '중국' 의식 형성
- '화하(華夏/化夏)': 문화적 통일성을 공유하는 나라들의 자기 의식

### 3. 중국과 '화이사상(華夷思想)'
- **화이사상**: 천하의 중심에 높은 군주가 있고, 주변은 동심원적으로 이미지로 세계를 파악
- 중화 = 문명, 이적(夷狄) = 야만 → **포용성**: 이적의 예를 사용하면 이적에 넣고, 중국에 들어오면 중국으로 봄
- **주권국가 세계상과의 차이**:
  - 주권국가: 각 국가가 고유 문화·주권·확실한 경계
  - 화이사상: 국경보다 군주 중심의 동심원적 지배, '천하'를 지배

### 4. 근대 내셔널리즘과 '중국' 개념
- **량치차오(梁啓超)**: 1901년 「중국사서론(中國史敍論)」, 「음빙실문집」
  - '중국사'라는 명칭 자체가 **100년 전에야** 사용되기 시작
  - '프랑스', '일본' 등과 나란히 세계국가의 하나로서 중국을 보고자 함
- 근대 이전: '대청(大淸)' 같은 왕조 이름이 곧 나라 이름, '중국'이라는 국가명칭 없었음

## 제2장: 중국문명의 형성 (pp. 29-41)

### 1. 황허문명과 초기 왕조
- 사마천 「사기」: 황제(黃帝) → 전욱 → 제곡 → 요 → 순 → **우(禹)=하(夏)왕조**
- 하왕조 → 은주(殷周)혁명 → **하분기(夏本紀)**: 폭군으로 교체되는 패턴
- **상(商=殷)나라**: 기원전 1600~1046년경, 황하 유역 통치
  - **갑골문(甲骨文)**: 거북 등껍질·소 어깨뼈에 새긴 점술문 = 최초의 문자 기록
  - 은허(殷墟, 안양시): 수만 점 청동·도기·옥 유물 발굴

### 2. 주(周)나라와 봉건제
- **봉건제(封建制)**: 영토를 나누어 제후에게 세습적으로 통치 → 종교적 권위로 통합
- 주나라의 수도 호경(鎬京) → 동쪽 낙읍으로 천도 = **춘추시대** 시작 (BC770)
- **춘추시대(BC770~BC403)**: 주왕의 권위 약화, 제후들 경쟁
- **전국시대(BC403~BC221)**: 본격적 영토국가 경쟁

### 3. 춘추전국시대의 변화 (4가지)
1. **농업생산 향상**: 철기·우경 보급 → 소농민 자립 → 씨족제도/봉건 정치제도 붕괴
2. **군현제 강화**: 부국강병 경쟁 → 관료제·군대 직접 통솔
3. **광역적 문화교류**: 활발해짐
4. **'중국'과 '화하' 개념의 확대**

### 4. 진(秦)의 통일 (BC221)
- 전국 7웅(제·초·진·연·한·위·조) 중 **서쪽 변경의 진**이 통일
- 법가 **상앙(商鞅)**의 개혁: 대가족→소가족, 군공에 따른 작위
- **진시황의 통일 정책**:
  - '황제(皇帝)' 칭호 최초 사용 (왕+제)
  - **군현제** 전국 확대: 36개 군(후에 48군)으로 나눔
  - 도량형·문자·화폐 통일, 도로 정비
  - 만리장성 축조

### 5. 봉건제 vs 군현제
| | 봉건제 | 군현제 |
|---|---|---|
| 지배 방식 | 영토 분할, 세습 | 황제가 직접 관리 임명 |
| 등장 | 주나라 | 진·한 이후 |
| 특징 | 간접 통치 | 직접 통치 |
- 실제로는 **혼용**: 한나라 초기 '군국제(郡國制)' = 군현+봉건 병존
- 중국의 전통적 국가관: 군현통치와 봉건통치를 섞어 **'천하'를 지배**, 국경선은 중요하지 않음

## 제3장: 농경사회와 유목사회 (pp. 42-53)

### 1. 흉노와 진·한 제국
- **BC9세기경**: 유라시아 초원지대에 청동제 마구와 무기를 가진 기마유목민 등장
- **스키타이** → 초원 서부, **흉노** → 중국 서북방에서 등장
- **묵특선우(冒頓單于)**: BC3세기 말, 동호·월지를 멸망시키고 초원지대 통일
  - 진시황 사후 혼란을 틈타 급속 성장
- **진시황**: 장군 몽염을 파견하여 흉노 격퇴, 만리장성 축조

### 2. 한제국과 흉노
- **한 고조 유방**: BC200 **백등산 전투** 패배 → 포위당해 간신히 탈출
  - 이후 **화친 약조**: 매년 비단·술·쌀 등을 공납, 공주 시집보냄
- **7대 무제(武帝, BC141~BC87)**: 적극적 정책 전환
  - **장건(張騫)** 서역 파견: 대월지와 동맹 시도 → 10년간 포로 → 귀환
  - 위청·곽거병 장군으로 흉노 공격, **하서회랑(河西回廊)** 4군 설치(돈황·무위·장액·주천)
  - 둔전(屯田): 병사를 보내 경작, 군사시설(봉화대 등) 구축
  - 한반도에 **한사군(漢四郡)** 설치 → 식민지적 통치
  - 오아시스 제국으로 진출 → 흉노의 수입 상실 → 내분, 분열

### 3. 사기(史記)의 흉노 묘사
- "물과 풀을 따라 이동, 성곽과 농업이 없다"
- "문자가 없으며 말로 약속을 한다"
- **중행열(中行說)과의 논쟁**: 흉노의 풍속을 변호
  - 한나라 사신: "흉노는 노인을 경시하지 않는가?"
  - 중행열: "전쟁에서 젊은이가 싸우고, 음식을 먼저 먹게 하는 것이 노인을 위하는 것"
  - 흉노 사회의 내적 논리를 보여주는 장면

### 4. 무제의 시대와 경제정책
- 군현제 + 봉건제 혼용 → **군국제(郡國制)**
  - 한 초기: 군현 + 봉건 병존, 제후국이 독립국처럼 행동
  - 무제: 제후왕의 세력 약화 정책(추은령 등)
- **경제정책**: 소금·철 **전매제**, **균수법(均輸法)·평준법(平準法)**
  - 물가 안정·재정 확보 목적
  - 「염철론(鹽鐵論)」: 관료 vs 학자·상인의 논쟁 기록

### 5. 후한(後漢)의 정치
- **왕망(王莽)**: AD8 신(新)나라 건국, 유교적 복고 개혁 → 실패
  - '주례(周禮)'에 근거한 제도 개혁, 호족·상인 통제
- **유수(劉秀, 광무제)**: AD25 후한 건국
- **천명관(天命觀)**: 천(天)의 덕이 있는 사람에게 천명을 준다 → 악정 시 천명 이동 = 역성혁명의 논리
- 후한 말: 환관·외척 전횡 → 184년 **황건의 난** → 분열

## 제4장: 분열과 다원화 (pp. 54-66)

### 1. 중국의 분열과 북방민족의 진출
- 4~5세기: 유라시아 초원지대에서 유목민 대이동
  - 유럽: 게르만 민족 이동 → 서로마 멸망
  - 동아시아: **5호16국** 시대
- 황건의 난 후: 위(조조·조비) → 촉(유비) → 오(손권) = **삼국시대**
- **사마염(무제)**: 진(晉) 통일(265~316) → 8왕의 난 → 분열

### 2. 5호16국과 호한체제
- **5호(五胡)**: 흉노·선비·갈·저·강
- 5세기 전반: **북위(北魏, 선비족 탁발부)**가 화북 통일
  - 한인관료 채용, 중국식 관제·제사·행정 정비
  - 중국식 제사와 함께 4월 하늘에 제사하는 **유목민 전통적 의례도 병행**

### 3. 북위 효문제(孝文帝)의 한화정책
- **낙양 천도**: 평성(군사 거점)에서 낙양(중원 중심지)으로
- **급진적 한화정책**:
  - 선비어 사용 금지, 호복 폐지
  - 한인 귀족과 선비 귀족 **혼인** 장려
  - 탁발 씨를 **원(元) 씨**로 개성(改姓)
- **반발**: 군인들의 반란(육진의 난 등), 결국 동서 분열
  - '한화'인가 '문화적 뒤처진 것의 이전'인가 → 다민족적 국가의 제도와 종속 선택

### 4. 육조정권과 강남개발
- **강남 이주**: 화북 전란 → 대규모 인구 남하
  - 강남 토박이들의 '정착민(正籍)' vs 북방 이주민 '백적(白籍/僑民)'
  - **토단법(土斷法)**: 호적 구분 해소, 이주민을 현지에 정착시키는 정책
- **남조 사회**: 선주민족(산월山越 등)과 한족의 융합 과정
  - 오늘날 중국 동남지역 방언 차이의 기원
- **남·북조**: 각각 독자적 문화 발전, '중화'의 새로운 형태 탄생

### 5. 위진남북조의 책봉관계
- 조공/책봉이 동아시아 **국제질서의 핵심**
- 신흥 국가들: 중국왕조로부터 **정통성 승인** 받으려 함
- **한반도**:
  - **고구려**: 313년 낙랑·대방 축출, 광개토왕·장수왕 시대 남북조에 조공
  - **백제**: 해외로 남조와 조공책봉 관계
  - **신라**: 6세기 한반도 남부 장악, 남북조와 교섭 시작
- **왜(일본)**: 위나라에 조공하여 '친위왜왕(親魏倭王)' 칭호 받음
  - 5세기 왜 5왕: 남조에 조공, **사지절도독제군사** 관작 요구 (군사적 지배권)
  - 한반도와 일본의 국제관계는 중국 양국 관계뿐 아니라 **동아시아 전체 국제질서**와 관련

### 6. 동아시아 세계론
- **니시지마 사다오(西嶋定生)**: 동아시아 국가들을 개별적이 아닌 '동아시아 세계' 속에서 파악
  - 책봉관계가 국제질서 부여
  - 하지만 **조공/책봉이 없던 시대에도 경제·문화 교류는 존재** (교재 마지막 문장)

## 시험 출제 가능 포인트

### 약술형 예상 질문

1. **화이사상(華夷思想)과 주권국가 세계상의 차이를 설명하시오.**
2. **봉건제와 군현제의 차이와 역사적 변천을 서술하시오.**
3. **한 무제의 대흉노 정책과 그 결과를 설명하시오.**
4. **진시황의 통일 정책(군현제, 문자·도량형 통일 등)의 의의를 서술하시오.**
5. **북위 효문제의 한화정책의 내용과 그에 대한 반발을 설명하시오.**
6. **위진남북조 시대 조공/책봉 관계의 의미와 한반도 국가들의 참여를 서술하시오.**
7. **중행열(中行說)의 논쟁이 보여주는 흉노 사회의 내적 논리를 설명하시오.**
8. **'중국'이라는 개념의 형성 과정과 근대 내셔널리즘과의 관계를 서술하시오.**

### T/F 판별 가능 사실들

1. **T** '중국'이라는 말은 기원전 10세기 무렵 「시경」에 처음 등장한다.
2. **F** 중화인민공화국에는 한족만 거주한다. → 56개 민족(한족 + 55개 소수민족)
3. **T** 화이사상에서는 이적이 중화의 예를 사용하면 중국으로 편입될 수 있다고 봄 (포용성).
4. **F** '중국사'라는 명칭은 고대부터 사용되었다. → 량치차오가 1901년에야 본격적으로 사용.
