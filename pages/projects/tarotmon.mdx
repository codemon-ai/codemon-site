# 🔮 타로몬

AI가 당신의 고민에 맞는 타로 카드를 뽑아드립니다.

> **서비스 URL**: [tarot.codemon.ai](https://tarot.codemon.ai)

## 서비스 소개

타로몬은 AI 기반 타로 리딩 서비스입니다.
사용자의 질문을 분석하고, 적절한 타로 카드를 선택하여 해석을 제공합니다.

### 주요 기능

- **질문 입력**: 고민이나 궁금한 점을 자유롭게 입력
- **AI 카드 선택**: 질문에 맞는 타로 카드 자동 선택
- **맞춤형 해석**: 질문 맥락을 반영한 개인화된 해석
- **히스토리**: 과거 리딩 기록 저장 및 조회

## 기술 스택

| 분류 | 기술 |
|------|------|
| Frontend | Next.js 14, TypeScript, Tailwind CSS |
| AI | OpenAI GPT-4, Claude API |
| Backend | Next.js API Routes |
| Database | Supabase (PostgreSQL) |
| Deployment | Vercel |

## 아키텍처

```
User Input → OpenAI Analysis → Card Selection → Interpretation Generation → Display
```

### AI 프롬프트 엔지니어링

- 질문 분류 및 키워드 추출
- 카드-질문 매칭 알고리즘
- 컨텍스트 기반 해석 생성

## 개발 배경

타로 리딩 서비스의 접근성을 높이고, AI를 활용해 일관되고 의미 있는 해석을 제공하고자 개발했습니다.
