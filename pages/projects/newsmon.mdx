# 📰 뉴스몬

AI 기반 개인 맞춤형 뉴스 큐레이션 서비스

> **상태**: 🟡 개발 중

## 서비스 소개

뉴스몬은 AI를 활용하여 사용자의 관심사에 맞는 뉴스를 선별하고
핵심 내용을 요약해서 제공하는 서비스입니다.

### 주요 기능

- **맞춤 큐레이션**: 관심 분야 기반 뉴스 선별
- **AI 요약**: 긴 기사를 핵심만 추출하여 요약
- **중복 제거**: 같은 이슈의 여러 기사 통합
- **감성 분석**: 기사의 논조 분석 (긍정/부정/중립)
- **트렌드 분석**: 시간대별 이슈 트렌드 시각화

## 기술 스택

| 분류 | 기술 |
|------|------|
| Frontend | Next.js 14, TypeScript, Tailwind CSS |
| AI | OpenAI GPT-4, Embeddings |
| Backend | Next.js API Routes, Python |
| Database | Supabase, Pinecone (Vector DB) |
| Crawling | Python (BeautifulSoup, Scrapy) |
| Deployment | Vercel, AWS Lambda |

## 아키텍처

```
News Sources → Crawler → AI Processing → Categorization → Personalized Feed
```

### AI 파이프라인

- 뉴스 수집 및 전처리
- 임베딩 기반 유사 기사 클러스터링
- LLM 기반 요약 및 분석

## 개발 배경

정보의 홍수 속에서 중요한 뉴스를 놓치지 않으면서도 시간을 효율적으로 사용하고 싶은 분들을 위해,
AI가 뉴스를 읽고 정리해주는 서비스를 기획했습니다.
