## 기술 특징

모든 프로젝트는 다음 원칙을 따릅니다:

1. **실제 서비스 운영**: 단순 포트폴리오가 아닌 실제 사용자를 위한 서비스
2. **AI 기술 활용**: LLM, 이미지 생성 등 최신 AI 기술 적용
3. **모던 스택**: Next.js, TypeScript, Tailwind CSS 기반
4. **확장 가능한 설계**: 서버리스 아키텍처, 마이크로서비스 지향
