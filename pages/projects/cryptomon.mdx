# 🪙 크립토몬

AI 기반 암호화폐 매매 시그널 서비스

> **상태**: 🟡 개발 중

## 서비스 소개

크립토몬은 AI를 활용하여 암호화폐 시장을 분석하고
매매 시그널을 제공하는 서비스입니다.

### 주요 기능

- **시장 분석**: 주요 코인 실시간 시세 및 기술적 분석
- **AI 시그널**: 머신러닝 기반 매매 타이밍 추천
- **온체인 분석**: 지갑 이동, 거래소 입출금 모니터링
- **뉴스 분석**: 암호화폐 관련 뉴스 감성 분석
- **알림 서비스**: 설정 조건 충족 시 실시간 알림

## 기술 스택

| 분류 | 기술 |
|------|------|
| Frontend | Next.js 14, TypeScript, Tailwind CSS |
| AI/ML | Python, TensorFlow, scikit-learn |
| Backend | FastAPI, Node.js |
| Database | PostgreSQL, Redis, InfluxDB |
| Data | Binance API, CoinGecko, Glassnode |
| Deployment | AWS (EC2, Lambda, SageMaker) |

## 아키텍처

```
Market Data → Feature Engineering → ML Models → Signal Generation → User Notification
```

### 분석 파이프라인

- 실시간 가격 데이터 수집
- 기술적 지표 계산 (RSI, MACD, 볼린저 밴드 등)
- 온체인 데이터 분석
- ML 모델 추론 및 시그널 생성

## 개발 배경

암호화폐 시장은 24시간 운영되며 변동성이 큽니다.
AI를 활용하여 시장을 지속적으로 모니터링하고,
데이터 기반의 매매 결정을 지원하고자 개발을 시작했습니다.
