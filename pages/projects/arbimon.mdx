# 📊 아비몬

거래소 간 차익거래 기회 탐지 서비스

> **상태**: 🟡 개발 중

## 서비스 소개

아비몬은 여러 거래소 간의 가격 차이를 실시간으로 모니터링하여
차익거래 기회를 탐지하고 알려주는 서비스입니다.

### 주요 기능

- **실시간 모니터링**: 다중 거래소 가격 동시 추적
- **차익 계산**: 수수료, 전송 시간 고려한 실제 수익률 계산
- **알림 시스템**: 설정 조건 충족 시 즉시 알림
- **히스토리**: 과거 차익거래 기회 분석
- **시뮬레이션**: 가상 거래를 통한 전략 검증

## 기술 스택

| 분류 | 기술 |
|------|------|
| Frontend | Next.js 14, TypeScript, Tailwind CSS |
| Backend | Node.js, Python |
| Database | Redis, PostgreSQL |
| Data | 거래소 WebSocket API |
| Deployment | AWS (EC2, Lambda) |

## 아키텍처

```
Exchange APIs → Price Aggregator → Arbitrage Calculator → Alert Service → User
```

### 핵심 로직

- 저지연 가격 데이터 수집
- 거래소별 수수료 및 출금 시간 반영
- 실행 가능성 기반 기회 필터링

## 개발 배경

암호화폐 시장의 비효율성을 활용한 차익거래는 저위험 수익 기회를 제공합니다.
개인이 수동으로 모니터링하기 어려운 다중 거래소 가격을 자동으로 추적하는 시스템을 구축하고자 합니다.
