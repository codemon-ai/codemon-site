# 🍽️ 슬림몬

AI 기반 간헐적 단식 관리 서비스

> **상태**: 🟢 운영 중

## 서비스 소개

슬림몬은 간헐적 단식을 쉽고 효과적으로 관리할 수 있도록 도와주는 AI 서비스입니다.
개인의 생활 패턴과 목표에 맞춘 맞춤형 단식 스케줄을 제안합니다.

### 주요 기능

- **단식 타이머**: 실시간 단식/식사 시간 추적
- **맞춤 스케줄**: 16:8, 20:4, 5:2 등 다양한 단식 패턴 지원
- **AI 코칭**: 개인 상황에 맞는 조언 및 동기부여
- **진행 기록**: 단식 이력 및 성과 시각화
- **알림 설정**: 단식 시작/종료 리마인더

## 기술 스택

| 분류 | 기술 |
|------|------|
| Frontend | Next.js 14, TypeScript, Tailwind CSS |
| AI | OpenAI GPT-4 |
| Backend | Next.js API Routes |
| Database | Supabase (PostgreSQL) |
| Deployment | Vercel |

## 아키텍처

```
User Schedule → Timer Service → AI Analysis → Personalized Coaching → Notification
```

### 핵심 기능

- 사용자 생활 패턴 분석
- 최적 단식 시간대 추천
- 지속적인 동기부여 메시지 생성

## 개발 배경

간헐적 단식의 효과는 널리 알려졌지만, 꾸준히 실천하기 어렵습니다.
AI를 활용해 개인 맞춤형 코칭을 제공하여 지속 가능한 단식 습관 형성을 돕고자 개발했습니다.
