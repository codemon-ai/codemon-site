# 💰 머니몬

AI 기반 투자 분석 및 포트폴리오 관리 서비스

> **상태**: 🟢 운영 중

## 서비스 소개

머니몬은 AI를 활용하여 투자 의사결정을 지원하는 서비스입니다.
시장 데이터 분석, 종목 스크리닝, 포트폴리오 최적화를 제공합니다.

### 주요 기능

- **시장 분석**: 국내외 주식, ETF 시장 동향 분석
- **종목 스크리닝**: 조건에 맞는 종목 자동 필터링
- **AI 인사이트**: 뉴스, 실적, 기술적 지표 종합 분석
- **포트폴리오 관리**: 자산 배분 및 리밸런싱 제안
- **알림 서비스**: 관심 종목 가격/뉴스 알림

## 기술 스택

| 분류 | 기술 |
|------|------|
| Frontend | Next.js 14, TypeScript, Tailwind CSS |
| AI | OpenAI GPT-4, Claude API |
| Backend | Next.js API Routes, Python |
| Database | Supabase (PostgreSQL) |
| Data | Yahoo Finance, KRX API |
| Deployment | Vercel, AWS Lambda |

## 아키텍처

```
Market Data → Data Pipeline → AI Analysis → Signal Generation → User Dashboard
```

### 데이터 파이프라인

- 실시간 시장 데이터 수집
- 뉴스 및 공시 크롤링
- 재무제표 분석

## 개발 배경

개인 투자자가 기관 수준의 분석 도구를 사용하기 어렵다는 점에 착안하여,
AI를 활용해 누구나 쉽게 접근할 수 있는 투자 분석 도구를 만들고자 개발했습니다.
