# BlueMango v2 — 패키지 상품 (복합 상품)

> 여러 개별 상품(호텔+골프+차량 등)을 **하나의 패키지로 묶어 할인 판매**하는 기능을 추가합니다.

<DocSection title="배경 — 왜 필요한가" accent="sky">

### 현재 (AS-IS)

- 고객이 "방콕 골프 3박4일"을 원하면 → 호텔, 골프장, 차량을 **각각 따로 예약/결제**
- 패키지 할인을 원하면 → **전화/카카오톡으로 문의** → 관리자가 수동 견적 → 개별 예약 생성
- 관리자는 구성 상품 가격을 일일이 계산하여 합산 후 할인 적용

### 문제점

| 문제 | 영향 |
|------|------|
| 패키지 할인을 시스템에서 적용 불가 | 수동 견적 → 관리자 업무 부담, 응대 지연 |
| 고객이 구성 상품을 일일이 찾아야 함 | 예약 과정이 복잡, 이탈률 증가 |
| 패키지 매출 추적 불가 | 어떤 패키지가 인기인지 데이터 없음 |
| 패키지 바우처를 수동 생성 | 구성 상품별 바우처를 개별 작성 |

### 목표 (TO-BE)

- "방콕 골프 3박4일" 같은 **패키지를 하나의 상품으로 등록**
- 고객은 패키지 상세 페이지에서 **한 번에 예약/결제**
- 관리자는 패키지 생성 화면에서 **구성 상품 선택 + 할인가 설정**

</DocSection>

<DocSection title="패키지 상품 구성" accent="emerald">

### 패키지 = 메인 상품 + 구성 상품들

| 구성 요소 | 설명 | 예시 |
|----------|------|------|
| 메인 상품 (Package) | 패키지 전체를 대표하는 상품 | 방콕 골프 3박4일 |
| 구성 상품 (PackageItem) | 패키지에 포함된 개별 상품들 | 호텔 3박, 골프 2라운드, 공항픽업, 골프장 이동 |

### 예시: "방콕 골프 패키지"

| # | 구성 상품 | 상품 타입 | 개별 가격 |
|---|----------|----------|----------|
| 1 | 방콕 시내 호텔 3박 | HOTEL | 450,000원 |
| 2 | 골프장 A 라운드 | GOLF | 200,000원 |
| 3 | 골프장 B 라운드 | GOLF | 180,000원 |
| 4 | 공항 ↔ 호텔 픽업 | VEHICLE | 80,000원 |
| 5 | 호텔 ↔ 골프장 이동 (2회) | VEHICLE | 120,000원 |
| | **개별 합산** | | **1,030,000원** |
| | **패키지 가격** | | **899,000원 (13% 할인)** |

</DocSection>

<DocSection title="관리자 경험" accent="violet">

### 패키지 상품 생성 절차

1. **상품 관리 → 패키지 상품 생성** 메뉴 진입
2. 패키지 기본 정보 입력 (이름, 설명, 대표 이미지, 카테고리)
3. **구성 상품 선택**: 기존에 등록된 상품 중 원하는 것을 추가
4. **패키지 가격 설정**: 개별 합산가 대비 패키지 판매가 입력
5. 저장 → 고객 사이트에 패키지 상품 노출

### 패키지 예약 관리

- **예약 목록**에서 패키지 예약은 "패키지" 뱃지로 구분
- **예약 상세**에서 구성 상품별 상태를 개별 관리 가능
  - 예: 호텔은 바우처 발행 완료, 골프는 아직 확인 중
- **부분 취소**: 구성 상품 중 일부만 취소 가능 (예: 골프 1회만 취소)

### 패키지 바우처

- 구성 상품별 **개별 바우처** 자동 생성
- 각 바우처는 해당 공급처 정보 포함
- 패키지 전체 일정을 보여주는 **종합 바우처**도 제공 (선택사항)

</DocSection>

<DocSection title="주문(Order) 시스템과의 관계" accent="sky">

| 구조 | 설명 |
|------|------|
| 패키지 1개 | = Order(주문) 1개 |
| 구성 상품 N개 | = Booking(예약) N개 |
| 결제 | = Payment 1개 (패키지 전체 금액) |

### 예시 구조

```
Order #1234 "방콕 골프 패키지"
├── Booking #A: 호텔 3박 (VOUCHER_SENT)
├── Booking #B: 골프 A 라운드 (PAYMENT_COMPLETED)
├── Booking #C: 골프 B 라운드 (PAYMENT_COMPLETED)
├── Booking #D: 공항 픽업 (VOUCHER_SENT)
├── Booking #E: 골프장 이동 (VOUCHER_SENT)
└── Payment #P1: 899,000원 (COMPLETED)
```
