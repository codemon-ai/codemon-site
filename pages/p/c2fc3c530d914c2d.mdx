## 시안 A — 선라이트 매거진

### 컨셉
Monocle / Kinfolk 매거진 스타일을 PRD 무드에 맞게 재해석. 큰 세리프 타이포그래피와 넓은 여백으로 글에 집중하는 격조 있는 디자인. 프레시 그린 카테고리 태그와 부드러운 카드 그리드.

### 특징
- 큰 세리프 히어로 타이틀 — 첫인상에서 전문성 전달
- 2열 카드 그리드 — 매거진 표지처럼 정돈된 글 목록
- 프레시 그린 카테고리 태그 — 시각적 분류와 액센트
- 넉넉한 여백 — 읽는 사람이 쉬어갈 공간

### 페이지 구성

**📄 홈페이지** — 큰 타이포 히어로 + 카테고리 카드 그리드
![시안 A 홈페이지](/images/blog-mockups/A1_sunlight_home.png)

**📋 콘텐츠 목록** — 카테고리 필터 필 + 2열 카드 그리드
![시안 A 콘텐츠](/images/blog-mockups/A2_sunlight_content.png)

**📖 글 상세** — 넓은 리딩 컬럼 + 그린 풀쿼트 박스
![시안 A 글 상세](/images/blog-mockups/A3_sunlight_detail.png)

### 이런 분께 추천

- 깔끔하고 격조 있는 느낌을 선호하는 분
- "덜어내는 디자인"을 좋아하는 분

## 시안 B — 보태니컬 노트

### 컨셉
식물과 자연에서 영감을 받은 부드러운 디자인. 보태니컬 일러스트레이션과 리프 아이콘이 곳곳에 배치되어 자연 친화적이고 따뜻한 느낌. 카테고리별 식물 아이콘으로 분류.

### 특징
- 보태니컬 일러스트 — 자연스러운 리프 패턴과 장식
- 리프 카테고리 아이콘 — 각 카테고리마다 식물 모티프
- 소프트 그린 카드 — 호버 시 부드러운 #CFE9DC 배경
- 뉴스레터 구독 — 하단 구독 폼으로 자연스러운 전환

### 페이지 구성

**📄 홈페이지** — 보태니컬 히어로 + 리프 아이콘 카드
![시안 B 홈페이지](/images/blog-mockups/B1_botanical_home.png)

**📋 콘텐츠 목록** — 좌측 카테고리 사이드바 + 3열 카드 그리드
![시안 B 콘텐츠](/images/blog-mockups/B2_botanical_content.png)

**📖 글 상세** — 리프 장식 타이틀 + 콜아웃 박스 + 관련글 + 뉴스레터
![시안 B 글 상세](/images/blog-mockups/B3_botanical_detail.png)

### 이런 분께 추천
- 자연/식물 테마의 따뜻한 블로그를 원하는 분
- 카테고리별 비주얼 분류를 좋아하는 분
- 구독자 전환이 중요한 분

## 시안 C — 모닝 스튜디오

### 컨셉
밝은 아침 스튜디오의 자연광 느낌. 극도로 미니멀한 레이아웃에 자연 사진과 일러스트가 조화를 이루는 현대적 디자인. Masonry 그리드로 비주얼 임팩트.

### 특징
- Masonry 카드 그리드 — 다양한 높이의 카드로 시각적 풍부함
- 자연광 배경 — 밝은 스튜디오 분위기의 UI
- 미니멀 리스트 뷰 — 콘텐츠 목록을 인덱스 스타일로 깔끔하게
- 브레드크럼 네비게이션 — 명확한 위치 표시

### 페이지 구성

**📄 홈페이지** — 미니멀 타이틀 + Masonry 카드 그리드 + 구독 버튼
![시안 C 홈페이지](/images/blog-mockups/C1_morning_home.png)

**📋 콘텐츠 목록** — 검색바 + 카테고리 탭 + 미니멀 리스트
![시안 C 콘텐츠](/images/blog-mockups/C2_morning_content.png)

**📖 글 상세** — 브레드크럼 + 센터 타이틀 + 인사이트 박스 + 공유 버튼
![시안 C 글 상세](/images/blog-mockups/C3_morning_detail.png)

### 이런 분께 추천
- 트렌디하고 현대적인 디자인을 선호하는 분
- 이미지와 글이 함께 어우러지는 블로그를 원하는 분
- 검색과 탐색이 편한 구조를 원하는 분

## 3가지 시안 비교

| | 시안 A (선라이트 매거진) | 시안 B (보태니컬 노트) | 시안 C (모닝 스튜디오) |
|---|---|---|---|
| **느낌** | 격조 있는 매거진 | 자연 친화적, 따뜻한 | 현대적, 밝은 공간 |
| **강점** | 글 중심, 읽기 좋음 | 비주얼 분류, 구독 전환 | 시각적 임팩트, 검색 |
| **레이아웃** | 2열 카드 그리드 | 사이드바 + 3열 | Masonry + 리스트 |
| **적합** | 에세이/칼럼형 | 라이프스타일/자연 | 다양한 주제/비주얼 |

## 공통 기술 스펙

| 항목 | 내용 |
|------|------|
| 프레임워크 | Next.js 15 + Tailwind CSS |
| 폰트 | Pretendard (본문) + Noto Serif KR (제목) |
| 호스팅 | Vercel (무료) |
| 블로그 엔진 | MDX (마크다운 기반) |
| 모드 | 라이트 온리 |

## 제작 과정

1. **시안 선택** — 위 3가지 중 하나 (또는 믹스)
2. **개발** — 1~2일 내 완성
3. **도메인 연결** — 원하는 도메인 구매 후 Vercel 연결
4. **콘텐츠 입력** — 마크다운으로 글 작성 → 자동 반영
5. **운영** — Vercel 무료 호스팅, 관리 비용 없음
