## 시안 A — 미니멀 매거진

**컨셉:** Monocle / Kinfolk 매거진처럼 큰 타이포그래피와 넓은 여백

![시안 A 히어로](/images/private/coach-blog-proposal/hero_abstract_a.png)

### 특징
- **큰 제목 타이포** — Serif 폰트로 격조 있는 첫인상
- **넓은 여백** — 콘텐츠에 집중, 읽는 사람이 쉬어갈 공간
- **카드형 그리드** — 블로그 목록이 매거진 표지처럼 보임
- **미니멀 네비게이션** — 상단에 홈 / 블로그 / 소개 / 연락만

### 페이지 구성
| 페이지 | 설명 |
|--------|------|
| 홈 | 큰 타이포 히어로 + 프로필 이미지 + 최근 글 3개 |
| 블로그 | 카테고리 필터 + 2열 카드 그리드 |
| 글 상세 | 넓은 본문 (max-w-2xl) + 양쪽 여백 |
| 소개 | 프로필 사진 + 경력/철학 텍스트 |
| 연락하기 | 이름, 이메일, 메시지 폼 |

### 이런 분께 추천
- 글 중심의 블로그를 원하는 분
- 깔끔하고 격조 있는 느낌을 선호하는 분
- "덜어내는 디자인"을 좋아하는 분

## 시안 B — 노트북 스타일

**컨셉:** Notion / Bear 앱처럼 좌측 사이드바 + 본문 중심의 깔끔한 필기장

![시안 B 히어로](/images/private/coach-blog-proposal/hero_workspace_b.png)

### 특징
- **좌측 사이드바** — 카테고리와 태그를 한눈에 (데스크탑)
- **타임라인 형태** — 날짜순으로 글이 흐르는 구조
- **읽기 모드** — 글 상세에서 큰 행간, 넉넉한 여백
- **태그 시스템** — #리더십 #소통 #코칭 등 자유로운 분류

### 페이지 구성
| 페이지 | 설명 |
|--------|------|
| 홈 | 좌측 사이드바(카테고리 + 미니 프로필) + 최신글 타임라인 |
| 블로그 | 타임라인 형태, 카테고리 필터 |
| 글 상세 | 읽기 모드 (max-w-xl, 큰 행간) |
| 소개 | 수첩에 적은 듯한 레이아웃 |
| 연락하기 | 미니멀 폼 |

### 이런 분께 추천
- 글을 자주 쓰고 정리하고 싶은 분
- Notion이나 블로그 앱의 깔끔함을 좋아하는 분
- 카테고리/태그로 체계적 분류를 원하는 분

## 시안 C — 카드 포트폴리오

**컨셉:** 전문가 브랜딩 사이트 + 블로그. 서비스 소개와 글이 함께 있는 구조.

![시안 C 히어로](/images/private/coach-blog-proposal/hero_pattern_c.png)

### 특징
- **랜딩페이지형 홈** — 히어로 + 서비스 카드 + 최근 글 + 후기
- **서비스 소개** — 교육 프로그램 / 1:1 코칭 / 퍼실리테이션 3개 카드
- **참여자 후기** — 실제 후기 카드로 신뢰 구축
- **CTA 버튼** — "문의하기"로 자연스럽게 연결

### 홈페이지 구조 (원페이지 스크롤)
| 섹션 | 내용 |
|------|------|
| 히어로 | 프로필 + "다정한 전문가, OOO입니다" + CTA |
| 서비스 소개 | 3개 카드 (교육/코칭/퍼실리테이션) |
| 최근 글 | 3개 블로그 카드 |
| 후기 | 참여자 후기 (이름, 한줄, 별점) |
| 연락하기 | CTA + 폼 |

### 이런 분께 추천
- 블로그뿐 아니라 서비스 홍보도 필요한 분
- 방문자에게 "이 사람은 전문가구나" 느낌을 주고 싶은 분
- 문의/상담 연결이 중요한 분

## 3가지 시안 비교

| | 시안 A (매거진) | 시안 B (노트북) | 시안 C (포트폴리오) |
|---|---|---|---|
| **느낌** | 격조 있는 매거진 | 깔끔한 필기장 | 전문가 브랜딩 |
| **강점** | 글 중심, 읽기 좋음 | 정리/분류 용이 | 서비스 홍보 + 블로그 |
| **적합** | 에세이/칼럼형 | 꾸준한 글쓰기 | 비즈니스 연결 |
| **레이아웃** | 중앙 정렬 + 여백 | 사이드바 + 본문 | 섹션 스크롤 |

---

## 제작 과정

1. **시안 선택** — 위 3가지 중 하나 (또는 믹스)
2. **개발** — 1~2일 내 완성
3. **도메인 연결** — 원하는 도메인 구매 후 Vercel 연결
4. **콘텐츠 입력** — 마크다운으로 글 작성 → 자동 반영
5. **운영** — Vercel 무료 호스팅, 관리 비용 없음
