# BlueMango v2 — 정보 구조 (IA)

> 작성일: 2026-02-27 | 대상: COO, PM | 문서 유형: PRD

## 1. 한 줄 요약

BlueMango v2는 **고객 사이트 42개 페이지**(14개 카테고리 상품 탐색 → 예약 → 결제 → 바우처)와 **어드민 사이트 35개 화면**(8개 메뉴 그룹)으로 구성되며, v1 대비 어드민 화면 수를 절반 이하로 줄이면서 장바구니·일정관리·정산 등 신규 기능을 추가합니다.

---

## 2. 고객 사이트 구조

### 2.1 하단 탭 (GNB)

고객 사이트의 하단에 항상 노출되는 4개 탭입니다.

| 순서 | 탭 | 설명 |
|:---:|------|------|
| 1 | **홈** | 메인 화면. 추천 상품, 카테고리 바로가기, 프로모션 배너 |
| 2 | **검색** | 카테고리별 상품 검색 및 필터링 |
| 3 | **일정** | 예약한 여행 일정을 캘린더로 관리 (v2 신규) |
| 4 | **마이페이지** | 예약 내역, 포인트, 프로필, 알림 등 개인 메뉴 |

### 2.2 상품 카테고리 (14개)

| # | 카테고리 | # | 카테고리 |
|:---:|------|:---:|------|
| 1 | 호텔 | 8 | 해양투어 |
| 2 | 풀빌라 | 9 | 가이드 |
| 3 | 골프텔 | 10 | 스파/마사지 |
| 4 | 골프장 | 11 | 레스토랑 |
| 5 | 차량 | 12 | 쇼/입장권 |
| 6 | 단독투어 | 13 | 공항패스트트랙 |
| 7 | 지상투어 | 14 | 비자 |

### 2.3 주요 사용자 흐름

고객이 상품을 발견하고 바우처를 받기까지의 핵심 흐름입니다.

```
홈/검색 → 카테고리 선택 → 상품 목록 → 상품 상세
  → 옵션 선택 → 장바구니 담기 (v2 신규)
    → 주문서 작성 → 결제 → 예약 확정 → 바우처 수령
```

**v2에서 달라진 점:**

- **장바구니/주문/결제 흐름 추가** — v1은 상품 하나씩 바로 예약 → v2는 여러 상품을 장바구니에 담아 한번에 결제 가능
- **일정관리 신규** — 예약 확정된 상품이 자동으로 캘린더에 반영, 여행 전체 일정을 한눈에 확인
- **공개 인보이스/바우처** — 고유 링크로 로그인 없이 인보이스·바우처 확인 가능 (카카오톡 등으로 공유 가능)

### 2.4 마이페이지 구성

| 메뉴 | 설명 |
|------|------|
| 예약관리 | 진행 중/완료/취소 예약 내역 조회 및 상세 확인 |
| 포인트 | 보유 포인트 조회, 적립/사용 내역 |
| 프로필 | 개인정보 수정, 소셜 계정 연동 관리 |
| 알림 | 예약 상태 변경, 프로모션 등 알림 목록 |
| 공지사항 | 서비스 공지, 업데이트 안내 |
| 여행팁 | 태국 여행 관련 유용한 정보 콘텐츠 |

## 3. 어드민 사이트 구조

### 3.1 메뉴 구성 (8개 그룹)

v1에서 80개 이상이던 화면을 v2에서 **35개로 통합**했습니다. 중복 메뉴를 제거하고, 필터 기반 통합 화면으로 전환한 결과입니다.

| # | 메뉴 그룹 | 주요 기능 | 비고 |
|:---:|------|------|------|
| 1 | **대시보드** | 오늘의 예약, 매출 요약, 주요 지표 | |
| 2 | **예약관리** | 예약 목록(상태 필터), 예약 상세, 결제 관리 | v1 11개 → 3개 |
| 3 | **상품관리** | 통합 상품 목록(타입 필터), 상품 등록/수정 | v1 7개 → 2개 |
| 4 | **회원관리** | 회원 목록, 회원 상세, 포인트 관리 | 포인트 관리 신규 |
| 5 | **콘텐츠관리** | 여행팁, 공지사항, 약관, Q&A, 리뷰 | v2 신규 |
| 6 | **정산관리** | 정산 내역, 정산 상세 | v2 신규 |
| 7 | **사이트관리** | 배너, 메인 구성, 카테고리 설정, 팝업 | |
| 8 | **시스템** | 관리자 계정, 알림 로그, 설정 | 알림 로그 신규 |

### 3.2 v1 → v2 주요 변경점

#### 예약관리: 11개 화면 → 3개

| v1 (기존) | v2 (변경) |
|------|------|
| 호텔 예약, 골프 예약, 차량 예약, 투어 예약 등 카테고리별 별도 메뉴 | **예약 목록 1개** — 상태·카테고리 필터로 통합 |
| 무통장입금 관리 3개 메뉴 (대기, 확인, 취소) | 제거 — 예약 목록 내 결제 상태 필터로 대체 |
| 차량예약 별도 2개 메뉴 | 제거 — 예약 목록에 통합 |
| 결제 관리 없음 | **결제 관리 신규** — 결제 내역 조회 및 환불 처리 |

#### 상품관리: 7개 화면 → 2개

| v1 (기존) | v2 (변경) |
|------|------|
| 호텔 상품, 골프 상품, 차량 상품, 투어 상품, 기타 상품, 항공 상품 등 별도 메뉴 | **상품 목록 1개** — 카테고리 필터로 통합 |
| 카테고리마다 다른 등록/수정 화면 | **상품 등록/수정 1개** — 카테고리에 따라 필드가 자동 전환 |
| 항공 상품 메뉴 (미사용) | 제거 |

#### 신규 추가

| 메뉴 | 설명 |
|------|------|
| 정산관리 | 파트너사별 정산 금액 집계 및 상세 내역 관리 |
| 콘텐츠관리 | 여행팁·공지·약관·Q&A·리뷰를 한곳에서 관리 |
| 포인트관리 | 회원별 포인트 적립/차감/내역 관리 |
| 알림 로그 | 발송된 알림(카카오, 이메일 등) 이력 조회 |
