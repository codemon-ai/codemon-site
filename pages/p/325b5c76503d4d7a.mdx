# BlueMango v2 릴리즈 노트

## 개요

- **기간**: 2026년 2월
- **커밋**: 169건
- **상태**: 개발 진행 중 (90%)
- **스택**: Next.js 15 + Turborepo + Drizzle ORM + Supabase + Better Auth

---

## 주요 변경 사항

### 🏗️ 인프라 & 초기 설정

- **Turborepo 모노레포 구축**: 7개 패키지 (`apps/web`, `apps/admin`, `packages/shared`, `packages/ui`, `packages/db`, `packages/auth`, `packages/email`)
- **DB 스키마 완성**: Drizzle ORM + Supabase Pooler, 29개 테이블 push 완료
- **인증 시스템**: Better Auth + Drizzle 어댑터, 카카오/네이버/구글 소셜 로그인, 비밀번호 재설정 이메일 플로우
- **이메일**: React Email 템플릿 + Resend 연동
- **v1→v2 마이그레이션**: 실제 스키마 기반 마이그레이션 스크립트, dry-run 20,448행 0에러, 라이브 마이그레이션 완료
- **시드 데이터**: 프로덕션 DB 정리 + 100상품/300예약 시드
- **PoC 검증**: Toss 웹훅 멱등성, react-pdf Vercel Serverless 호환, v1 bcrypt 해시 better-auth 호환 검증

### 🖥️ 어드민 (Admin)

- **대시보드 & 통계**: KPI 카드, 칸반 보드, 매출 차트, 상품/예약/회원 통계
- **상품 관리 (5개 타입별 동적 폼)**:
  - 골프: 요금 CRUD, 상세 폼, 골프텔 필터
  - 호텔/풀빌라: 객실 관리 + 가용성 캘린더, 어메니티/부대시설, 조식/교통정보
  - 차량: 인보이스/미팅장소, 렌탈종류, 편도·왕복 가격 분리
  - 투어/기타: 상세 관리 폼
- **예약/결제 관리**: 예약 목록/상세/상태 변경, 무통장 입금 확인 + 환불 UI
- **카테고리 관리**: 4단계 계층 CRUD 트리, 순서 이동(▲▼)
- **기타 관리**: 공급사, 정산, 회원, 콘텐츠, 설정, 환율 UI 일괄 구현
- **UX 개선**: SearchableSelect 드롭다운, 마크다운 에디터, 약관 콘텐츠 시딩

### 🌐 프론트엔드 (Web)

- **디자인 시스템**: v1 디자인 토큰 + tailwind typography, 다크 테마 홈페이지 리디자인
- **홈/검색**: 히어로 배너 캐러셀, 무한 스크롤, 이중 통화(바트+원) 표시, 한국어 검색 + 다중 단어 OR 조건
- **카테고리/상품 상세**: 카테고리 메뉴, 상품 카드(평점/리뷰수/영문명), 접이식 섹션, HTML 렌더링
- **예약 위자드 (5개 타입)**:
  - 차량: 3단계 위자드 (가격 계산 + 제출)
  - 골프텔: 예약 위자드
  - 단독투어: 상세 + 예약 위자드
  - 풀빌라: 상담 전환
  - 기타: 예약 위자드
- **장바구니/결제**: DB 기반 장바구니, GNB 뱃지, 쿠폰/포인트 체크아웃 연동, 결제 성공 페이지
- **마이페이지**: 예약 내역, 포인트, 프로필 수정, 비밀번호 변경, 3스텝 회원가입
- **기타**: 리뷰 작성/삭제, Q&A, 알림, PDF 생성(인보이스/바우처 + QR), Google Maps, 정적 페이지, SEO 메타데이터

### 🔧 버그 수정

- 카카오 소셜 로그인 세션 유지 (쿠키 prefix, scope, OAuth 콜백 — 다수 회차 수정)
- 상품 detail JSONB 이중 인코딩 대응
- 차량 예약 wizard searchParams 유실, serviceType DB 수정
- 호텔 예약 차단 날짜 검증 + 할증 가격 반영
- 어드민 통계 탭 전환 시 stale data crash
- 빌드/배포: unused 패키지 제거, typecheck 수정, 의존성 추가

### 🧪 테스트

- **E2E 테스트 스위트**:
  - 웹 코어 플로우 E2E (홈 → 검색 → 상세 → 예약)
  - 예약 폼/장바구니/체크아웃 E2E
  - 차량 예약 E2E (어드민 + 웹)
  - 웹 전체 페이지 종합 E2E
  - 어드민 전체 섹션 종합 E2E
  - 결제 플로우 E2E
  - 인보이스/바우처 이메일 통합 테스트
- **상품 타입별 어드민 E2E**: 골프/호텔/풀빌라/차량 각각 TC 매트릭스
- **프로덕션 검증**: 어드민 UX 17TC, 통계 13TC, 카테고리 10TC, 이중통화 5TC PASS
