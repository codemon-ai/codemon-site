# 비공개 페이지 시스템 테스트

이것은 비공개 페이지 시스템의 초기 설정 테스트 페이지입니다.

## 특징

- ✅ URL 직접 접근 가능: `/p/test-initial-setup`
- ✅ 사이드바/네비게이션에 노출되지 않음
- ✅ 마크다운 렌더링 정상 작동
- ✅ 다크모드 지원

## 코드 예제

```typescript
// 간단한 TypeScript 예제
const message: string = "비공개 페이지가 정상 작동합니다!";
console.log(message);
```

## 다음 단계

```

```
