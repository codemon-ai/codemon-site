## 현재 진행 상태

### 완료 (Wave 0+1+2)
- 프로젝트 설정 (Turborepo, Supabase, Drizzle)
- 인증 (Better Auth + 카카오/구글/네이버)
- 상품 관리 (14개 카테고리 전체 — 호텔, 풀빌라, 골프텔, 골프장, 차량, 투어 6종, 스파, 레스토랑, 쇼/입장권, 비자, 공항패스트트랙)
- 예약/결제 플로우 (토스페이먼츠)
- 어드민 (상품/예약/회원/정산/콘텐츠/설정 관리)
- 인보이스/바우처 웹 페이지
- 쿠폰/포인트 시스템
- E2E 테스트 (Playwright, 전체 상품 타입)

### 남은 작업 (3대 미구현)
1. **Order(주문) 도입** — 장바구니 통합 결제, 부분 취소
2. **PG 전환** — 토스 → 이니시스 (카드사 심사 대기 중)
3. **알림 시스템** — 이메일 정상화, 알림톡 복원, PDF 생성
