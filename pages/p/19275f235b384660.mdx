# 검색 (Search) 플로우차트

> IA 항목: SRCH-01 ~ SRCH-10 | 총 10개 화면

## 플로우차트

```mermaid
flowchart TD
    GNB_SEARCH["🔍 GNB 검색 탭<br/>GNB-02"] --> SEARCH_PAGE

    SEARCH_PAGE["검색 페이지 진입<br/>/search<br/>SRCH-01"]

    SEARCH_PAGE --> NO_KEYWORD{"검색어 유무?"}

    NO_KEYWORD -->|없음| SUGGESTIONS["인기 검색어<br/>최근 검색어<br/>SRCH-02"]
    NO_KEYWORD -->|있음| SEARCH_EXEC["검색 실행"]

    SUGGESTIONS -->|키워드 클릭| SEARCH_EXEC

    SEARCH_EXEC --> HAS_RESULT{"검색 결과?"}

    HAS_RESULT -->|있음| RESULT_LIST["검색 결과 목록<br/>2열 레이아웃<br/>SRCH-03"]
    HAS_RESULT -->|없음| EMPTY_STATE["검색 결과 없음<br/>빈 상태 안내<br/>SRCH-04"]

    RESULT_LIST --> FILTER["카테고리 필터<br/>SRCH-05"]
    FILTER -->|해제| FILTER_RESET["전체 결과 복원<br/>SRCH-06"]
    FILTER_RESET --> RESULT_LIST

    RESULT_LIST --> SCROLL["무한 스크롤<br/>추가 로딩<br/>SRCH-08"]
    SCROLL --> RESULT_LIST

    RESULT_LIST -->|상품 클릭| PRODUCT_DETAIL["상품 상세<br/>/package/{id}<br/>SRCH-07"]

    SEARCH_PAGE --> CATEGORY_GRID["카테고리 그리드<br/>/categories<br/>SRCH-09"]
    CATEGORY_GRID -->|아이콘 클릭| CATEGORY_LIST["카테고리 목록<br/>/category/{slug}<br/>SRCH-10"]

    style SEARCH_PAGE fill:#1E3A5F,stroke:#1E3A5F,color:#fff
```
