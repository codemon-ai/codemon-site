# BlueMango v2 — 기능 명세서

> **작성일**: 2026-02-27
> **대상**: COO, PM
> **문서 유형**: PRD (기능 명세)

---

## 한 줄 요약

BlueMango v2는 태국 현지 여행 상품(호텔, 골프, 차량, 투어 등 14개 카테고리)을 한국 고객에게 제공하는 **예약 플랫폼**입니다. v1 대비 장바구니, 결제 통합관리, 정산, 콘텐츠 관리 등 **핵심 신규 기능**이 추가되며, 어드민 메뉴를 대폭 간소화하여 **운영 효율**을 높입니다.

---

## 고객 사이트 기능

<DocSection title="홈 / 검색" accent="sky">

고객이 사이트에 처음 방문했을 때 보는 화면과 상품을 찾는 기능입니다.

| 기능 | 설명 |
|------|------|
| 홈 화면 | 추천 상품, 인기 카테고리, 프로모션 배너 노출 |
| 키워드 검색 | 상품명, 지역명 등으로 자유 검색 |
| 카테고리 필터 | 14개 카테고리 중 원하는 항목 선택 |
| 지역 필터 | 방콕, 파타야, 치앙마이 등 지역별 검색 |

</DocSection>

<DocSection title="상품 조회" accent="sky">

고객이 여행 상품을 살펴보는 기능입니다. 총 **14개 카테고리**를 **5가지 상품 타입**으로 구분합니다.

| 상품 타입 | 포함 카테고리 |
|-----------|--------------|
| 숙박 | 호텔, 풀빌라, 골프텔 |
| 골프 | 골프장 |
| 차량 | 차량 (픽업, 렌트 등) |
| 투어 | 단독투어, 지상투어, 해양투어, 가이드 |
| 기타 서비스 | 스파/마사지, 레스토랑, 쇼/입장권, 공항패스트트랙, 비자 |

**이용 흐름:** 카테고리 목록에서 상품 선택 → 상품 상세 확인 → 예약 진행

</DocSection>

<DocSection title="예약 / 결제" accent="sky">

고객이 상품을 선택하고 결제하는 과정입니다.

| 기능 | 설명 | 비고 |
|------|------|------|
| 장바구니 | 여러 상품을 담아두고 한 번에 결제 | **v2 신규** |
| 예약 위자드 | 날짜 → 인원 → 옵션 순서로 단계별 선택 | |
| 카드 결제 | 신용카드/체크카드 온라인 결제 | |
| 무통장입금 | 계좌이체 후 입금 확인 | |
| 가상계좌 | 1회용 계좌 발급 후 입금 | |
| 비회원 예약 | 회원가입 없이 예약 가능 | |

</DocSection>

<DocSection title="마이페이지" accent="sky">

로그인한 고객이 자신의 정보와 예약을 관리하는 공간입니다.

| 기능 | 설명 |
|------|------|
| 예약 관리 | 내 예약 목록 조회, 상태별 필터 (예약확정/결제대기/완료 등) |
| 예약 상세 | 개별 예약의 세부 내용 확인 |
| 포인트 | 적립/사용 내역 확인 |
| 프로필 | 회원정보 수정 (이름, 연락처, 비밀번호 등) |
| 알림 | 예약 확정, 결제 완료 등 주요 알림 확인 |
| 공지사항 / 여행팁 | 사이트 공지 및 태국 여행 정보 열람 |

</DocSection>

<DocSection title="인보이스 / 바우처" accent="sky">

예약 확정 후 발행되는 문서입니다. 고객과 현지 업체 모두 확인할 수 있습니다.

| 기능 | 설명 | 비고 |
|------|------|------|
| 공개 링크 | 로그인 없이 URL만으로 접근 가능 (카카오톡 등으로 공유) | |
| PDF 다운로드 | 인보이스/바우처를 PDF 파일로 저장 | **v2 추가 예정** |

</DocSection>

<DocSection title="콘텐츠" accent="sky">

고객에게 유용한 정보를 제공하는 페이지입니다.

| 기능 | 설명 |
|------|------|
| 여행팁 | 태국 여행에 필요한 정보 (입국, 환전, 교통 등) |
| 공지사항 | 서비스 관련 안내 |
| 이용약관 / 개인정보처리방침 | 법적 고지 문서 |

</DocSection>

## 어드민 기능

<DocSection title="대시보드" accent="violet">

관리자가 로그인 후 가장 먼저 보는 화면입니다. 핵심 지표를 한눈에 파악할 수 있습니다.

| 항목 | 설명 |
|------|------|
| 매출 요약 | 일별/주별/월별 매출 현황 |
| 예약 수 | 기간별 예약 건수 |
| 신규 회원 | 신규 가입자 수 |
| 상태별 예약 현황 | 결제대기, 예약확정, 완료, 취소 등 상태별 건수 |
| 인기 상품 | 예약이 많은 상품 순위 |

</DocSection>

<DocSection title="예약 관리" accent="violet">

모든 예약을 조회하고 처리하는 기능입니다. v1에서 11개로 분산되어 있던 메뉴를 **3개로 통합**하여 운영 편의성을 높였습니다.

| 기능 | 설명 | 비고 |
|------|------|------|
| 예약 목록 | 전체 예약 조회 (상태/상품타입/결제방법/기간으로 필터) | v1의 11개 메뉴 통합 |
| 예약 상세 | 예약 내용 확인, 상태 변경, 인보이스/바우처 발행 | |
| 예약 생성 | 관리자가 전화/카카오 접수 건을 직접 등록 | |

</DocSection>

<DocSection title="결제 관리 (v2 신규)" accent="violet">

결제와 환불을 한 곳에서 관리합니다. v1에서는 무통장입금 확인이 별도 메뉴에 있었으나 v2에서 통합됩니다.

| 기능 | 설명 | 비고 |
|------|------|------|
| 결제 목록 | 전체 결제 내역 조회 | **v2 신규** |
| 환불 처리 | 취소/환불 요청 처리 | **v2 신규** |
| 무통장입금 확인 | 입금 확인 및 예약 상태 변경 | v1 별도 메뉴에서 통합 |

</DocSection>

<DocSection title="상품 관리" accent="violet">

판매 상품을 등록하고 수정하는 기능입니다. v1에서 7개로 분산되어 있던 메뉴를 **2개로 통합**했습니다.

| 기능 | 설명 | 비고 |
|------|------|------|
| 통합 상품 목록 | 모든 상품을 한 화면에서 조회 (타입/카테고리/상태 필터) | v1의 7개 메뉴 통합 |
| 상품 등록/수정 | 상품 타입에 따라 입력 항목이 자동 변경 | |
| 상품 분류 관리 | 4단계 카테고리 구조 관리 | |

**상품 타입별 관리 항목:**

| 상품 타입 | 주요 관리 항목 |
|-----------|---------------|
| 숙박 (호텔/풀빌라/골프텔) | 객실 종류, 날짜별 가용성 캘린더 |
| 골프 | 코스 정보, 그린피 (이용료) |
| 차량 | 차종, 서비스 종류 (공항 픽업, 렌트 등) |
| 투어 | 일정, 포함 사항, 불포함 사항 |
| 기타 서비스 | 운영시간, 소요시간 |

</DocSection>

<DocSection title="회원 관리" accent="violet">

가입 회원의 정보와 등급을 관리합니다.

| 기능 | 설명 |
|------|------|
| 회원 목록/상세 | 전체 회원 조회, 개별 회원 상세 정보 확인 |
| 등급/역할 관리 | 7단계 권한 체계 (일반 → 직원 → 관리자 → 최고관리자) |
| 포인트 관리 | 포인트 수동 지급/차감, 변동 이력 조회 |

</DocSection>

<DocSection title="정산 관리 (v2 신규)" accent="violet">

현지 공급업체(호텔, 골프장 등)에 대한 정산을 관리합니다.

| 기능 | 설명 | 비고 |
|------|------|------|
| 월별 정산 | 공급업체별 월 단위 정산 내역 조회 | **v2 신규** |
| 정산 승인/지급 | 정산 내역 확인 후 승인 및 지급 처리 | **v2 신규** |
| 공급업체 관리 | 업체 정보, 수수료율 등록/수정 | **v2 신규** |

</DocSection>

<DocSection title="콘텐츠 관리 (v2 신규)" accent="violet">

사이트에 노출되는 각종 콘텐츠를 관리합니다.

| 기능 | 설명 | 비고 |
|------|------|------|
| 여행팁 | 여행 정보 글 작성/수정/삭제 | **v2 신규** |
| 공지사항 | 서비스 공지 작성/수정/삭제 | **v2 신규** |
| 약관 관리 | 서비스 이용약관, 개인정보처리방침, 취소/환불 규정 | **v2 신규** |
| 상품 문의 (Q&A) | 고객 문의에 대한 답변 관리 | **v2 신규** |
| 리뷰 관리 | 고객 리뷰 삭제/숨김 처리 | **v2 신규** |

</DocSection>

<DocSection title="쿠폰 / 포인트" accent="violet">

고객 혜택을 설정하고 관리합니다.

| 기능 | 설명 |
|------|------|
| 포인트 정책 설정 | 적립률, 사용 조건 등 정책 설정 |
| 수동 지급/차감 | 특정 고객에게 포인트 직접 부여 또는 차감 |
| 사용 이력 | 전체 포인트 적립/사용 내역 조회 |

</DocSection>

<DocSection title="사이트 설정" accent="violet">

사이트 전반의 운영 설정을 관리합니다.

| 기능 | 설명 |
|------|------|
| 일반 설정 | 사이트명, 로고, 대표 연락처 등 |
| 메뉴/네비게이션 | 상단 메뉴 구성, 순서 변경 |
| 검색엔진 최적화 (SEO) | 페이지별 제목, 설명, 키워드 관리 |
| 환율 관리 | 원화(KRW) ↔ 바트(THB) 환율 설정 |

</DocSection>
