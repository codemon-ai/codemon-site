# Claude 마스터클래스

## 개요

| 항목 | 내용 |
|------|------|
| 제목 | Claude 마스터클래스 — Chat부터 Code까지, AI 7시간 완전 정복 |
| 형식 | 강의 + 라이브 데모 + 핸즈온 실습 (7시간) |
| 대상 | AI를 제대로 배우고 싶은 누구나 (비개발자 ~ 개발자) |
| 강사 | 코드몬 (codemon.ai) |

## 이런 분에게 추천합니다

- AI를 써보고 싶은데 **어디서부터 시작해야 할지** 모르겠는 분
- ChatGPT는 써봤지만 **"그래서 뭐가 달라졌어?"** 상태인 분
- **바이브 코딩**으로 직접 서비스를 만들어보고 싶은 분
- AI를 **업무에 진짜로 적용**하고 싶은 직장인
- Claude의 **채팅·코워크·코드** 전체를 한 번에 배우고 싶은 분

## 이 강의를 들으면

> 7시간 후, AI가 "신기한 장난감"에서 **"매일 쓰는 업무 도구"**로 바뀝니다.

- AI가 어떻게 동작하는지 **원리를 이해**합니다
- Claude 채팅으로 **문서 분석, 크롤링, 이미지 분석**을 직접 해봅니다
- Claude 코워크로 **데이터 분석, 뉴스 자동화, 기획서 작성**을 경험합니다
- Claude 코드로 **실제 웹사이트 3개를 만들어 배포**합니다
- 터미널에서 **딥서칭, 도식화, 분석 보고서 자동 생성**을 합니다
- 나만의 **AI 활용 로드맵**을 가지고 돌아갑니다

| 세션 | 내용 | 시간 |
|------|------|------|
| 1-1 | **최신 AI 뉴스 & 트렌드** — Claude Cowork 출시 충격(SW 주가 413조 증발), Rakuten 코드 자율 관리, Lovable 비개발자 앱 개발, 한국 기업 85% AI 도입 | 10분 |
| 1-2 | **업계 리더들의 인사이트** — 빌 게이츠, 샘 올트먼, 젠슨 황, 손정의가 말하는 AI의 미래 | 8분 |
| 1-3 | **AX가 필요한 이유** — DX(디지털 전환)를 넘어 AX(AI 전환)로, "할 것인가"가 아니라 "어떻게 잘 할 것인가" | 7분 |

### Part 3. Claude 생태계 — 채팅·코워크·코드 (15분)

Claude의 세 가지 모드를 이해하고, 각각 언제 쓰는지 감을 잡습니다.

| 모드 | 특징 | 적합한 업무 |
|------|------|------------|
| **채팅 (Chat)** | 대화형 AI, 파일 업로드, 이미지 분석 | 질문, 분석, 번역, 글쓰기 |
| **코워크 (Cowork)** | 협업형 AI, 프로젝트 관리, 예약 기능 | 데이터 분석, 문서 작성, 자동화 |
| **코드 (Code)** | 개발자용 AI, 파일 시스템 접근, MCP | 웹사이트 제작, 자동화, 딥서칭 |

> 이 세 모드를 7시간에 걸쳐 전부 실습합니다. 오늘이 끝나면 언제 뭘 쓸지 스스로 판단할 수 있습니다.

### Part 4. Claude 채팅 실습 (75분)

Claude 채팅으로 할 수 있는 것들을 직접 해봅니다. 4가지 실습, 4가지 능력.

| 세션 | 내용 | 시간 |
|------|------|------|
| 4-1 | **사이트 정보 크롤링** — URL 하나로 웹사이트 정보 수집·분석·경쟁사 비교까지 | 20분 |
| 4-2 | **PDF 보고서 완전 분석** — 100페이지짜리 보고서 업로드 → 5분 만에 핵심 요약 + 인사이트 추출 + 후속 질문 | 20분 |
| 4-3 | **이미지 분석 & 콘텐츠 생성** — 사진 한 장으로 블로그 글 + SNS 캡션 + 해시태그 + 상품 설명 동시 생성 | 20분 |
| 4-4 | **AI 개인 과외 선생님** — Claude를 나만의 튜터로 활용, Projects 기능으로 학습 컨텍스트 유지 | 15분 |

> **채팅만으로도 충분히 강력합니다.** 하지만 이건 시작일 뿐입니다.

☕ **쉬는 시간 (10분)**

| 5-2 | **데일리 뉴스 자동화** — 관심 분야 뉴스 큐레이션 + **예약 기능**으로 매일 아침 자동 발행 | 25분 |

> **코워크는 "AI 팀원"입니다.** 채팅이 대화라면, 코워크는 함께 일하는 것.
