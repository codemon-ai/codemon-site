## 이런 분에게 추천합니다

- AI를 써보고 싶은데, **어디서부터 시작할지** 모르는 대표
- 적은 인원으로 **큰 팀 못지않은 결과물**을 내고 싶은 대표
- AI 도입을 검토 중인데, **실제 사례와 비용**이 궁금한 대표

## 핵심 성과

> 1인이 AI 에이전트 7명과 함께 만든 실제 결과

- **월 $3 비용**으로 AI 개발팀 운영
- **40편+ 블로그**, 브랜드 사이트, 자동화 파이프라인을 AI 팀과 구축
- 리서치 → 글 → 이미지 → SNS 포스팅까지 **전 과정 자동화**

## 커리큘럼

### Part I. 지금 무슨 일이

#### M1. AI 시대, 무엇이 달라졌나 (15분)

숫자로 보는 AI 시대의 변화.

- AI 코딩 도구 시장: **$85억** (2026년) → $473억 (2034년)
- 멀티 에이전트 시스템 기업 문의: **작년 대비 15배 증가** (Gartner)
- 전문 개발자의 **62~85%**가 AI 코딩 도구 사용 중
- Google/MS 코드의 **30%가 AI 생성** (Gartner 전망: 연말 60%)

> "예전엔 AI가 비서였다면, 지금은 팀원이다.
> 일을 시키면 알아서 찾아보고, 실행하고, 결과를 보고한다."

#### M2. 대표가 알아야 할 AI, 몰라도 되는 AI (15분)

AI의 3단계 진화를 비개발자 언어로 설명합니다.

| 단계 | 비유 | 할 수 있는 것 |
|------|------|-------------|
| AI 챗봇 | 똑똑한 인턴 | 질문에 답변, 문서 요약, 아이디어 브레인스토밍 |
| AI 코딩 도구 | 실행력 있는 팀원 | 코드 작성, 파일 수정, 테스트 실행, 배포 |
| AI 팀 | 부서 하나 | 여러 AI가 역할 분담해서 프로젝트를 진행 |

**대표가 알아야 할 것**: AI가 무엇을 할 수 있는지, 비용은 얼마인지, 어디에 쓰면 효과적인지

**몰라도 되는 것**: 모델 구조, 파라미터 수, 토큰 계산법

> **데모 1 — 실시간 리서치**: NotebookLM으로 경쟁사 분석 보고서 2분 만에 생성

---

### Part II. 어떻게 작동하나

#### M3. AI가 일하는 방식 — 도구에서 팀원으로 (20분)

AI 에이전트가 어떻게 "지시 → 실행 → 보고"를 하는지 CEO 언어로 설명합니다.

**AI 연결 기술 (MCP)**

"AI가 회사 시스템에 로그인해서 직접 일하는 것"

- AI + Slack → 메시지를 읽고 답장
- AI + Google Calendar → 일정 확인하고 회의를 잡음
- AI + GitHub → 코드를 읽고, 수정하고, 배포

**AI 팀원 (에이전트)**

한 가지 역할을 전담하는 AI. 여러 에이전트가 팀을 이루면 부서가 됩니다.

```
대표 (지시)
├── AI 리서처: "경쟁사 분석해줘"
├── AI 에디터: "블로그 글 써줘"
└── AI 개발자: "랜딩페이지 만들어줘"
```

> **데모 2 — 회사 도구 연결**: MCP로 슬랙, 캘린더, 노션을 AI에 연결하는 과정

### Part III. 우리 회사에 적용

#### M4. 5명이 50명을 이기는 구조 (25분) ★

이 강의의 핵심 모듈. 실제 운영 중인 AI 팀 구조와 자동화 사례를 공개합니다.

**실제 AI 팀 구성**

| AI 에이전트 | 역할 | 하는 일 |
|------------|------|--------|
| 로디 | 퍼블리셔 | 블로그 발행, 이미지 생성, SNS 포스팅 |
| 누비 | 리서처 | 시장 조사, 경쟁사 분석, 트렌드 리포트 |
| 베어 | QA 엔지니어 | 테스트 자동화, 버그 리포트 |
| 뉴비 | 주니어 개발자 | 반복 코딩, 컴포넌트 생성 |
| 나래 | 기획자 | 요구사항 정리, 문서 작성 |
| 옥토 | DevOps | 배포 자동화, 모니터링 |
| 아울 | 코드 리뷰어 | PR 리뷰, 코드 품질 관리 |

> 이 팀의 월 운영 비용: **약 $3** (AI API 사용료)

**자동화 파이프라인 3가지**

> **데모 3 — 블로그 자동화**: 메모 하나가 블로그 글이 되는 과정 (9분 파이프라인)
>
> 메모 작성 → AI가 초안 생성 → 이미지 생성 → MDX 변환 → 사이트 배포 → SNS 공유

> **데모 4 — SNS 자동 포스팅**: 리서치 → 글 → 이미지 → 포스팅 전 과정 자동화

> **데모 5 — 바이브 코딩**: 코드 한 줄 없이 랜딩페이지 만들기

### Part IV. 믿을 수 있나

#### M5. 품질과 보안 — 신뢰할 수 있는 AI (10분)

"AI가 실수하면 어쩌죠?" — 가장 많이 받는 질문에 답합니다.

**품질 관리**
- AI가 만든 결과물을 다른 AI가 검증 (이중 검증 구조)
- 자동 테스트로 배포 전 문제 사전 차단

**보안**
- AI에게 최소 권한만 부여 (읽기만, 쓰기만 등)
- 민감 정보 접근 제한
- 자동 보안 점검으로 취약점 탐지

> **데모 6 — 자동 품질 점검**: AI가 사이트를 직접 테스트하고 버그 리포트를 작성하는 과정

### Part V. 시작하기

#### M6. 내일부터의 로드맵 (15분)

지금 당장 시작할 수 있는 단계별 가이드.

**1주차**: AI 챗봇으로 업무 보조 시작 (리서치, 문서 요약, 이메일 초안)

**1개월차**: 반복 업무 하나를 AI에 위임 (SNS 포스팅, 고객 응대 초안 등)

**3개월차**: AI 에이전트 2~3명으로 팀 구성, 파이프라인 자동화

**비용 가이드**

| 단계 | 월 비용 | 효과 |
|------|---------|------|
| AI 챗봇 활용 | $20~60 | 리서치/문서 작업 시간 50% 절감 |
| 업무 자동화 | $60~200 | 반복 업무 자동화, 1인 생산성 3배 |
| AI 팀 운영 | $200~500 | 5명이 50명 규모의 결과물 |

> **데모 7 — 창업 시뮬레이션**: 앞의 모든 도구를 엮어 에이전시를 세팅하는 전 과정
