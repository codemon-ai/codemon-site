## 커리큘럼 (3시간)

> 총 180분 = 강의 155분 + Q&A 25분 | 데모 7개 포함

### 1장. "지금 AI로 뭘 할 수 있는 건가요?" (45분)

#### M1. 413조원이 사라진 날 — AI가 바꾸고 있는 것 (25분)

2026.02.05, Claude Cowork 출시 → 미국 SW/금융/법률 주식 시총 **2,850억 달러(413조원) 증발**. "이건 단순한 도구 출시가 아니라, 일하는 방식의 전환점입니다."

- AI 코딩 도구 시장: **$85억** (2026년) → $473억 (2034년, CAGR 24%)
- 멀티 에이전트 시스템 기업 문의: **작년 대비 15배 증가** (Gartner)
- 전문 개발자의 **62~85%**가 AI 코딩 도구 사용 중
- Google/MS 코드의 **30%가 AI 생성** (Gartner 전망: 연말 60%)

기업 사례: Rakuten(1,250만 줄 코드 자율 관리), TELUS(50만 시간 절감), Zapier(800+ 에이전트 운영)

> MIT Technology Review: "Generative Coding" 2026 10대 돌파 기술 선정
> Claude Code: 출시 6개월 만에 $1B ARR 달성

**Before(2024)**: Tab 자동완성 → **After(2026)**: 지시 → 파일 검색 → 코드 수정 → 테스트 → 배포까지 자율 수행

> 에이전트가 인간 개입 없이 **연속 20개 액션을 자율 수행** (6개월 전 2배)

#### M2. 비개발자도 앱을 만든다 — 바이브 코딩의 시대 (20분)

**바이브 코딩**: 자연어로 원하는 것을 설명하면 AI가 전부 구현.

- 바이브 코딩 시장: 2026년 $47억 → 2027년 $123억 (CAGR 38%)
- 사용자의 **63%가 비개발자**
- Lovable: 8개월 만에 **$100M ARR** (역대 최고 속도 SaaS 성장)

스타트업에 의미: 프로토타이핑 비용 → 0, 가치가 "만드는 것"에서 "잘 만들고 운영하는 것"으로 이동

| 단계 | 비유 | 할 수 있는 것 |
|------|------|-------------|
| AI 챗봇 | 똑똑한 인턴 | 질문에 답변, 문서 요약, 아이디어 브레인스토밍 |
| AI 코딩 도구 | 실행력 있는 팀원 | 코드 작성, 파일 수정, 테스트 실행, 배포 |
| AI 팀 | 부서 하나 | 여러 AI가 역할 분담해서 프로젝트를 진행 |

> **데모 1 — 실시간 리서치**: NotebookLM으로 경쟁사 분석 보고서 2분 만에 생성

### 2장. "AI가 진짜 우리 회사 일을 할 수 있나요?" (40분)

#### M3. AI 팀원의 작동 원리 — 연결, 역할, 매뉴얼 (20분)

**MCP = "AI의 USB-C 포트"** — AI가 회사 시스템에 로그인해서 직접 일하는 표준 프로토콜 (1,000+서버, 빅테크 전원 채택)

- MCP + Slack → 메시지를 읽고 요약, 답장
- MCP + Google Calendar → 일정 확인하고 회의를 잡음
- MCP + GitHub → 코드를 읽고, PR을 만들고, 리뷰

**에이전트 = 역할이 있는 AI 팀원.** 여러 에이전트가 팀을 이루면 부서가 됩니다.

```
대표 (지시)
├── AI 리서처: "경쟁사 분석해줘"
├── AI 에디터: "블로그 글 써줘"
└── AI 개발자: "랜딩페이지 만들어줘"
```

**CLAUDE.md = AI 온보딩 문서**: 프로젝트 규칙, 금지 사항, 선호 방식을 한 번 적어두면 AI가 매번 따름

**스킬 = AI 업무 매뉴얼**: 반복 업무를 스킬로 정의하면 자동 인식 + 일관된 결과. 비개발자도 3분이면 생성 가능

> **데모 2 — 회사 도구 연결**: MCP로 슬랙을 AI에 연결하고 채널 요약 요청

#### M4. 5명이 50명을 이기는 구조 (20분) ★

이 강의의 핵심 모듈. 실제 운영 중인 AI 팀 구조와 자동화 사례를 공개합니다.

**실제 AI 팀 7명**

| AI 에이전트 | 역할 | 하는 일 | 월 비용 |
|------------|------|--------|---------|
| 로디 | PM & 퍼블리셔 | 기획, 블로그 발행, 이미지 생성, SNS | ~$0.5 |
| 누비 | QA & Dev | 코드 리뷰, 테스트, 디버깅 | ~$0.5 |
| 나래 | 리서처 | 시장 조사, 경쟁사 분석, 트렌드 | ~$0.5 |
| 베어 | 디자이너 | UI/UX, 컴포넌트, 프론트엔드 | ~$0.3 |
| 뉴비 | 주니어 개발자 | 반복 코딩, 컴포넌트 생성 | ~$0.3 |
| 옥토 | DevOps | 서버, API, 배포 자동화 | ~$0.3 |
| 아울 | 코드 리뷰어 | PR 리뷰, 코드 품질 관리 | ~$0.3 |

> 이 팀의 월 운영 비용: **약 $3** (AI API 사용료)

**자동화 파이프라인 진화**: v1(36분) → v2(13분) → v3(9분) — **75% 시간 단축**

현실적 시작점: 2봇 협업 (맥 2대, iCloud 공유 폴더, 비동기 통신)

> **데모 3 — 블로그 자동화**: 텔레그램 한 줄 → 리서치 → 작성 → 검수 → 배포 (9분 파이프라인)
>
> **데모 4 — SNS 자동 포스팅**: 블로그 발행 → 이미지 생성 → 플랫폼별 텍스트 → 자동 포스팅

### 3장. "AI가 실수하면 어쩌죠? 비용은요?" (40분)

#### M5. 품질 — 틀릴 수 있다, 그래서 시스템이 필요하다 (20분)

**대전제**: "AI는 틀린다. 문제는 어떻게 잡느냐."

**이중 검증 구조**: AI 생성 → AI 검증 → 사람 승인

**Hooks — 자동 가드레일**
- PreToolUse: 위험한 작업 사전 차단
- PostToolUse: 자동 포맷팅 + 테스트 실행
- 보안: 최소 권한 원칙 + 민감 정보 제한 + 데이터 미학습

> **데모 5 — 바이브 코딩**: 코드 한 줄 없이 랜딩페이지 만들기
>
> **데모 6 — 자동 품질 점검**: AI가 사이트를 테스트하고 버그 리포트를 작성하는 과정

#### M6. 비용 — 솔직한 숫자 (20분)

| 모델 | 비용 | 비유 |
|------|------|------|
| Opus 4.6 | $15 / $75 (입출력) | 시니어 개발자 |
| Sonnet 4.6 | $3 / $15 | 미드레벨 (가성비 최고) |
| Haiku 4.5 | $1 / $5 | 주니어 / 인턴 |

| 단계 | 월 비용 | 할 수 있는 것 |
|------|---------|-------------|
| 시작 | ~$20 | AI 챗봇, 서브에이전트 가끔 |
| 적극 활용 | $200~400 | Agent Teams 일상 사용, 병렬 작업 |
| 풀 파이프라인 | $500+ | 24시간 자동화 운영 |

> 개발자 1명 평균 하루 ~$6 | 신입 인건비의 **1/10 ~ 1/20**

비용 최적화: 모델 분리, 프롬프트 캐싱(90% 절감), Agent Teams 선별 사용

### 4장. "우리 회사는 어디서부터 시작하죠?" (30분)

#### M7. 내일부터의 로드맵 (30분)

**1주차**: AI 챗봇으로 업무 보조 시작 (리서치, 문서 요약, 이메일 초안) — $0~20

**1개월차**: 반복 업무 위임 + CLAUDE.md 작성 + 첫 스킬 만들기 — $20~100

**3개월차**: MCP 연결 + 에이전트 2~3명 운영 + 첫 파이프라인 — $100~400

**스타트업 유형별 시작점**: SaaS(피드백 분류), 커머스(상품 설명), 에이전시(리서치), 콘텐츠(발행 파이프라인)

**핵심**: "프롬프트 잘 쓰기"가 아니라 "시스템 잘 만들기" — CLAUDE.md + 스킬 + 훅

> **데모 7 — 창업 시뮬레이션**: 가상 스타트업을 세팅하는 전 과정 (CLAUDE.md → 스킬 → MCP → 자동화)
