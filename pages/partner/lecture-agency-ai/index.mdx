# AI 에이전트로 에이전시 업무 자동화하기

## 개요

| 항목 | 내용 |
|------|------|
| 제목 | AI 에이전트로 에이전시 업무 자동화하기 |
| 형식 | 발표형 강의 (1~2시간) |
| 대상 | 에이전시 회사 (대표/기획자 + 개발자 혼합) |
| 강사 | 길벗 (codemon.ai) + 진연화 |

## 목표

1. **에이전시 대표/PM**: AI 에이전트가 뭔지, 왜 지금 도입해야 하는지, 비용은 얼마인지 이해
2. **에이전시 개발자**: Claude Code 기반 멀티 에이전트 워크플로우 구성법 파악
3. **공통**: 내일부터 할 수 있는 구체적 액션 플랜 제공

## 청중 특성

- **비개발자 (대표/기획자/PM)**: AI 도구 경험은 있지만 코딩 에이전트는 낯설 수 있음
- **개발자**: 코딩은 하지만 AI 에이전트 활용은 초기 단계일 가능성
- **공통 관심사**: "우리 에이전시 업무를 자동화할 수 있나?", "비용은?", "얼마나 현실적인가?"
