# Module 5: 내일부터 할 수 있는 것 (10분)

> 발표 목표: 청중이 강의장을 나서면서 "오늘 저녁에 이것부터 해봐야지"라고 생각하게 만들기

---

## 레벨별 액션 플랜

### Lv.1 — 오늘 할 수 있는 것 (0원, 30분)

| 액션 | 방법 | 대상 |
|------|------|------|
| **Claude.ai 가입** | claude.ai 접속, 무료 플랜 시작 | 전체 |
| **업무 프롬프트 만들기** | 반복하는 업무를 Claude에게 시켜보기 | 전체 |
| **회의록 요약** | 회의 내용 붙여넣기 → 요약 + 액션 아이템 추출 | PM/대표 |
| **코드 리뷰 요청** | 코드 붙여넣기 → "이 코드 리뷰해줘" | 개발자 |

**핵심**: "오늘 하던 업무 중 하나를 Claude에게 시켜보세요. 그게 시작입니다."

---

### Lv.2 — 이번 주에 할 수 있는 것 ($20/월, 2시간)

| 액션 | 방법 | 대상 |
|------|------|------|
| **Claude Pro 가입** | 월 $20 → 더 많은 대화, Claude Cowork 접근 | 전체 |
| **Claude Code 설치** | `npm install -g @anthropic-ai/claude-code` | 개발자 |
| **CLAUDE.md 작성** | 프로젝트 루트에 팀 규칙/컨벤션 문서화 | 개발자 |
| **첫 스킬 만들기** | Cowork에서 반복 업무를 스킬로 정의 | PM/대표 |

```bash
# Claude Code 설치 (개발자용)
npm install -g @anthropic-ai/claude-code

# 프로젝트 폴더에서 실행
cd ~/my-project
claude
> 이 프로젝트 구조를 분석하고 CLAUDE.md를 만들어줘
```

**CLAUDE.md 작성 시작점**:
```markdown
# CLAUDE.md

## 프로젝트 개요
[프로젝트에 대한 한 줄 설명]

## 기술 스택
[사용 중인 기술 나열]

## 코딩 규칙
- [팀 컨벤션 1]
- [팀 컨벤션 2]

## 금지 사항
- 프로덕션 DB 직접 수정 금지
- main 브랜치 직접 push 금지
```

### Lv.3 — 이번 달에 할 수 있는 것 ($100~200/월, 주말 하루)

| 액션 | 방법 | 대상 |
|------|------|------|
| **서브에이전트 활용** | 코드 리뷰 자동화 (code-reviewer 서브에이전트) | 개발자 |
| **훅 설정** | 코드 수정 시 자동 테스트 + 알림 | 개발자 |
| **MCP 서버 연결** | Slack/GitHub MCP로 워크플로우 자동화 | 개발자/PM |
| **첫 파이프라인** | 간단한 2단계 자동화 (리서치 → 보고서) | 팀 전체 |
| **Agent Teams 체험** | 병렬 에이전트 작업 시도 | 개발자 |

## 에이전시 도입 로드맵

```
Month 1: 탐색
├── Claude.ai로 업무 프롬프트 실험
├── 팀 내 AI 챔피언 1명 선정
└── "어떤 업무를 자동화하면 좋을까?" 리스트업

Month 2: 기반 구축
├── Claude Code 도입 (개발팀)
├── CLAUDE.md 작성 (프로젝트별)
├── 첫 자동화 성과 공유
└── MCP 서버 1~2개 연결

Month 3: 확장
├── 서브에이전트 역할 정의
├── 반복 업무 스킬로 표준화
├── 파이프라인 구축 시작
└── 비용 대비 효과 측정 → 확대/축소 결정
```

## Q&A

### 예상 질문과 답변

**Q: "코드 모르는 사람도 할 수 있나요?"**
A: Lv.1은 누구나 가능합니다. Claude.ai에서 대화하고, Cowork에서 스킬을 만드는 건 코딩 경험 없이도 됩니다. Lv.2부터는 개발자의 도움이 필요합니다.

**Q: "보안은 괜찮나요? 우리 코드가 학습에 쓰이나요?"**
A: Claude는 API/Pro 플랜에서 사용자 데이터로 모델을 학습하지 않습니다. 엔터프라이즈 플랜은 추가 보안 보장이 있습니다.

**Q: "비용이 계속 늘어나지 않나요?"**
A: 모델 분리(Opus/Sonnet/Haiku)와 프롬프트 캐싱으로 최적화하면 개발자 1인 월 $100~200 수준으로 유지 가능합니다. ROI는 보통 첫 달부터 체감됩니다.

**Q: "우리 에이전시 규모(5~10명)에도 효과가 있나요?"**
A: 오히려 소규모 에이전시에 더 효과적입니다. 대기업은 도입 프로세스가 복잡하지만, 소규모 팀은 이번 주부터 시작할 수 있습니다.

**Q: "어떤 업무부터 자동화하면 좋나요?"**
A: 반복적이고, 구조화되어 있고, 실패해도 치명적이지 않은 업무부터. 추천: 경쟁사 리서치, 코드 리뷰, 주간 보고서 자동 생성.

## 마무리 메시지

> "완벽한 AI 시스템을 한 번에 만드는 게 아닙니다.
> 오늘 Claude에 가입하고, 내일 업무 하나를 시켜보고,
> 다음 주에 CLAUDE.md를 쓰면 됩니다.
> 기술 선택에서 중요한 건 '최적'이 아니라
> **'지금 동작하면서 내일 개선할 수 있는가'**입니다."
