# Module 1: 왜 지금 AI 에이전트인가 (15분)

> 발표 목표: 청중에게 "지금 안 하면 늦는다"는 위기감과 기회를 동시에 전달

---

## 오프닝 — 숫자가 말해준다

> "코드를 작성하는 사람에서, AI를 지휘하는 사람으로."

핵심 숫자 3개로 시작:

| 숫자 | 의미 |
|------|------|
| **$85억** | 2026년 AI 코딩 도구 시장 규모 (2034년 $473억, CAGR 24%) |
| **1,445%** | 멀티 에이전트 시스템 기업 문의 증가율 (Gartner) |
| **62~85%** | AI 코딩 도구 사용 중인 전문 개발자 비율 |

추가 임팩트 수치:
- 91% 기업이 최소 1개 AI 코딩 도구 도입
- MIT Technology Review: "Generative Coding"을 2026년 10대 돌파 기술로 선정
- Bloomberg: "AI 코딩 에이전트가 테크 업계에 생산성 패닉 유발"

---

## 자동완성 → 자율 에이전트 (패러다임 전환)

**Before (2024)**: Tab 누르면 몇 줄 완성 → 개발자가 검토 → 반영

**After (2026)**: "이 API에 페이지네이션 추가해줘" → 에이전트가 관련 파일 찾기 → 스키마 분석 → 코드 수정 → 테스트 작성 → CI 실행

> Anthropic 보고서: 에이전트가 인간 개입 없이 **연속 20개 액션을 자율 수행** (6개월 전 대비 2배)

**비개발자에게 설명할 때**: "예전엔 AI가 비서였다면, 지금은 팀원이다. 일을 시키면 알아서 찾아보고, 실행하고, 결과를 보고한다."

---

## 에이전시가 주목해야 하는 이유

### 에이전시의 특성 = AI 자동화의 적격 타겟

| 에이전시 특성 | AI 자동화 적합도 |
|-------------|----------------|
| 프로젝트마다 비슷한 업무 흐름 반복 | 패턴화된 작업 = 자동화 최적 타겟 |
| 리서치 → 기획 → 디자인 → 개발 → QA 파이프라인 | 단계별 에이전트 배치 가능 |
| 다수 프로젝트 병렬 운영 | 에이전트로 스케일 아웃 |
| 클라이언트 요구 변경 빈번 | 빠른 프로토타이핑으로 대응 |

**핵심 메시지**: "에이전시는 '같은 종류의 프로젝트를 반복'한다. AI 에이전트는 '반복적이고 구조화된 작업'에 가장 강하다. 완벽한 매칭이다."

## Claude Cowork 출시 임팩트

2026년 2월 5일, Anthropic이 Claude Opus 4.6 + Claude Cowork + Agent Teams를 동시 출시.

> 그날 미국 SW/금융/법률 주식 **시총 2,850억 달러(약 413조원) 증발** — 시장이 "이건 진짜 패러다임 전환이다"라고 반응한 것

주요 기업 사례:
- **Rakuten**: 1,250만 줄 코드베이스에서 7시간 자율 작업, 99.9% 정확도, 인간 코드 기여 0%
- **TELUS**: 13,000+ AI 솔루션 생성, 50만 시간 절감
- **Zapier**: 조직 전체 89% AI 채택, 800+ 에이전트 내부 운영
- **Claude Code**: 출시 6개월 만에 **$1B ARR** 달성

## 바이브 코딩 — 비개발자도 앱을 만든다

**바이브 코딩**: 자연어로 원하는 것을 설명하면 AI가 전부 구현

- 사용자의 **63%가 비개발자**
- **Lovable**: 8개월 만에 $100M ARR (역대 최고 속도 SaaS 성장)
- 바이브 코딩 시장: 2026년 $47억, 2027년 $123억 전망 (CAGR 38%)

**에이전시에 의미하는 것**:
- 클라이언트가 "이런 앱 만들어줘" 대신 직접 만들기 시작 → 에이전시의 가치가 "만들어주는 것"에서 **"잘 만들어주는 것 + 운영해주는 것"**으로 이동
- 프로토타이핑/PoC 단계를 에이전트가 담당 → 에이전시 인력은 고부가가치 작업에 집중
