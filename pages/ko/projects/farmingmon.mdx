# 🌾 파밍몬

앱테크(App+Tech) 리워드 수집 자동화 서비스

> **상태**: 🟡 개발 중

## 서비스 소개

파밍몬은 다양한 앱테크 서비스의 리워드를 효율적으로 수집하고
관리할 수 있도록 도와주는 자동화 서비스입니다.

### 주요 기능

- **리워드 통합 관리**: 여러 앱의 포인트/리워드 현황 통합 조회
- **자동 출석 체크**: 일일 출석 보상 자동 수령
- **미션 알림**: 수행 가능한 미션 알림
- **수익 분석**: 월별/연별 리워드 수익 분석
- **최적화 가이드**: 시간 대비 효율 높은 활동 추천

## 기술 스택

| 분류 | 기술 |
|------|------|
| Frontend | Next.js 14, TypeScript, Tailwind CSS |
| Backend | Node.js, Python |
| Automation | Playwright, Selenium |
| Database | PostgreSQL |
| Scheduler | Node-cron, AWS Lambda |
| Deployment | AWS (EC2, Lambda) |

## 아키텍처

```
Scheduler → Automation Scripts → App APIs → Reward Collection → Dashboard
```

### 자동화 흐름

- 스케줄 기반 작업 실행
- 앱별 자동화 스크립트 관리
- 결과 수집 및 리포팅

## 개발 배경

앱테크는 소소한 수익을 올릴 수 있지만, 매일 여러 앱을 체크하는 것이 번거롭습니다.
반복적인 작업을 자동화하여 최소한의 노력으로 리워드를 수집할 수 있는 시스템을 만들고자 합니다.
