# codemon.ai

Pro Vibe Coder & Cracked Engineer - 개인 브랜드 사이트 및 블로그

## 기술 스택

- **프레임워크**: Nextra 3 (Next.js 14)
- **콘텐츠**: MDX (마크다운)
- **스타일**: Nextra Docs Theme
- **배포**: Vercel

## 개발

```bash
# 의존성 설치
npm install

# 개발 서버 시작
npm run dev

# 프로덕션 빌드
npm run build
```

## 구조

```
pages/
├── index.mdx           # 홈
├── about.mdx           # 소개
├── projects/           # 프로젝트
├── blog/               # 블로그
└── docs/               # 문서
```
