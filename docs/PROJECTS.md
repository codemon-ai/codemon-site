## 운영 중

### 🔮 타로몬 (TarotMon)
- **설명:** AI 기반 타로 카드 리딩 서비스. 매일 오늘의 운세부터 연애, 재물, 커리어까지 다양한 스프레드 제공.
- **기술:** Next.js, OpenAI API
- **URL:** https://tarot.codemon.ai
- **상태:** ✅ 운영 중

### 💪 슬림몬 (SlimMon)
- **설명:** 부부가 함께 사용하는 간헐적 단식 관리 앱. 단식 타이머, 체중 기록, 통계 제공.
- **기술:** Next.js
- **URL:** https://slimmon.vercel.app
- **상태:** ✅ 운영 중

### 💵 달러시그널 (Dollar Signal)
- **설명:** 달러 환전 최적 타이밍을 알려주는 시그널 서비스. 환율 분석 + 텔레그램 알림.
- **기술:** Next.js, FastAPI, Telegram Bot
- **URL:** https://dollar-signal-web.vercel.app
- **상태:** ✅ 운영 중

### 💰 머니몬 (MoneyMon)
- **설명:** 부부가 함께 사용하는 종합 투자 분석 시스템. 포트폴리오 관리, 자산 배분 추적.
- **기술:** Next.js
- **URL:** https://moneymon.vercel.app
- **상태:** ✅ 운영 중

### 🎨 그림하루 (GrimHaru)
- **설명:** AI 그림일기 서비스. 일기를 작성하면 AI가 감정을 분석하고 그에 맞는 이미지를 생성.
- **기술:** Next.js, OpenAI API, DALL-E
- **URL:** https://grimharu-beta.vercel.app
- **상태:** ✅ 운영 중
